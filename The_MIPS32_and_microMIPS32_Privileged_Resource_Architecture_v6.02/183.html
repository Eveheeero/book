9.27 Count Register (CP0 Register 9, Select 0) MIPS® Architecture For Programmers Vol. III: MIPS32® / microMIPS32™ Privileged Resource Architecture, Rev. 6.02 183 9.27 Count Register (CP0 Register 9, Select 0) Compliance Level: Required. The Count register acts as a timer, incrementing at a constant rate, whether or not an instruction is executed, retired, or any forward progress is made through the pipeline. The rate at which the counter increments is implementation- dependent, and is a function of the pipeline clock of the processor, not the issue width of the processor. The Count register can be written for functional or diagnostic purposes, including at reset or to synchronize proces- sors. The Count register can also be read via RDHWR register 2.
