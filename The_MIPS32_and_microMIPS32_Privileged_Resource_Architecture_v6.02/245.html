9.48 Configuration Register 3 (CP0 Register 16, Select 3) MIPS® Architecture For Programmers Vol. III: MIPS32® / microMIPS32™ Privileged Resource Architecture, Rev. 6.02 245 9.48 Configuration Register 3 (CP0 Register 16, Select 3) Compliance Level: Required if any optional feature described by this register is implemented: Release 2 of the Architecture, the SmartMIPS™ ASE, or trace logic; Optional otherwise. The Config3 register encodes additional capabilities. All fields in the Config3 register are read-only. Release 5 adds Config3 LPA to allow software to determine the presence of XPA (>36-bit PA support).
