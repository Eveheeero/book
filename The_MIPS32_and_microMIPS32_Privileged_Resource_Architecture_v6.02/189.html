9.32 Status Register (CP Register 12, Select 0) MIPS® Architecture For Programmers Vol. III: MIPS32® / microMIPS32™ Privileged Resource Architecture, Rev. 6.02 189 9.32 Status Register (CP Register 12, Select 0) Compliance Level: Required. The Status register is a read/write register that contains the operating mode, interrupt enabling, and the diagnostic states of the processor. Fields of this register combine to create operating modes for the processor. See “MIPS32 and microMIPS32 Operating Modes” on page 21 for a discussion of operating modes, and “Interrupts” on page 71 for a discussion of interrupt modes.
