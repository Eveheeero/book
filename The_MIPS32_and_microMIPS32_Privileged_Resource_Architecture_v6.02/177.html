9.24 BadVAddr Register (CP0 Register 8, Select 0) MIPS® Architecture For Programmers Vol. III: MIPS32® / microMIPS32™ Privileged Resource Architecture, Rev. 6.02 177 9.24 BadVAddr Register (CP0 Register 8, Select 0) Compliance Level: Required. The BadVAddr register is a read-only register that captures the most recent virtual address that caused one of the fol- lowing exceptions: • Address error (AdEL or AdES) • TLB Refill • TLB Invalid (TLBL, TLBS) • TLB Modified The BadVAddr register does not capture address information for cache or bus errors, or for Watch exceptions, since none is an addressing error.
