9.63 CacheErr Register (CP0 Register 27, Select 0) MIPS® Architecture For Programmers Vol. III: MIPS32® / microMIPS32™ Privileged Resource Architecture, Rev. 6.02 297 9.63 CacheErr Register (CP0 Register 27, Select 0) Compliance Level: Optional. The CacheErr register provides an interface with the cache error detection logic that may be implemented by a pro- cessor. The exact format of the CacheErr register is implementation-dependent and not specified by the architecture. Refer to the processor specification for the format of this register and a description of the fields.
