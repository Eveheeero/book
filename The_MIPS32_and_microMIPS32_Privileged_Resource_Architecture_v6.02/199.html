9.33 IntCtl Register (CP0 Register 12, Select 1) MIPS® Architecture For Programmers Vol. III: MIPS32® / microMIPS32™ Privileged Resource Architecture, Rev. 6.02 199 9.33 IntCtl Register (CP0 Register 12, Select 1) Compliance Level: Required (Release 2). The IntCtl register controls the expanded interrupt capability added in Release 2 of the Architecture, including vec- tored interrupts and support for an external interrupt controller. This register does not exist in implementations of Release 1 of the Architecture. Figure 9.34 shows the format of the IntCtl register; Table 9.48 describes the IntCtl register fields. .
