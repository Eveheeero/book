If this bit is set in a TLB entry, an attempt to fetch an instruction or to load MIPS16 PC-rel- ative data from the virtual page causes a TLB Invalid or a TLBXI exception, even if the V (Valid) bit is set. The XI bit is writable only if the XIE bit of the PageGrain register is set. If the XIE bit of PageGrain is not set, the XI bit of EntryLo0/EntryLo1 is set to zero on any write to the register, regardless of the value written. This bit is optional and its existence is denoted by the Config3 RXI or Config3 SM register fields.
