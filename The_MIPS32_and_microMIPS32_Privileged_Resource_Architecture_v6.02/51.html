Further, there is a performance penalty in context-switching. Limitations on use of the EVA load/store instructions are as follows: • Only usable from Kernel execution mode. • Only usable on a memory segment configured with a User access mode (AM). • The address translation selected will be mapped if possible, else unmapped. More simply, a TLB based address translation is preferred. Refer to Volume II of the MIPS Architectural Reference manual for further information on the EVA Load/Store instructions. The availability of these instructions are indicated by the Config5 EVA register field. Table 4.6 lists kernel load/store instructions.
