TEQ Trap if Equal 416 The MIPS32Â® Instruction Set Manual, Revision 6.06 Format: TEQ rs, rt MIPS32 Purpose: Trap if Equal To compare GPRs and do a conditional trap. Description: if GPR[rs] = GPR[rt] then Trap Compare the contents of GPR rs and GPR rt as signed integers. If GPR rs is equal to GPR rt, then take a Trap excep- tion. The contents of the code field are ignored by hardware and may be used to encode information for system software. To retrieve the information, system software may load the instruction word from memory. Alternatively, if CP0 BadInstr is implemented, the code field may be obtained from BadInstr .
