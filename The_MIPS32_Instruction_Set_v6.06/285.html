In paired-single for- mat, both the halves of the pair are copied to fd. The move is non-arithmetic; it causes no IEEE 754 exceptions, and the FCSR Cause and FCSR Flags fields are not modified. Restrictions: The fields fs and fd must specify FPRs valid for operands of type fmt. If the fields are not valid, the result is UNPRE- DICTABLE. The operand must be a value in format fmt; if it is not, the result is UNPREDICTABLE and the value of the operand FPR becomes UNPREDICTABLE. The result of MOV.PS is UNPREDICTABLE if the processor is executing in the FR=0 32-bit FPU register model. It is predictable if executing on a 64-bit FPU in the FR=1 mode, but not with FR=0, and not on a 32-bit FPU. Availability and Compatibility: MOV.PS has been removed in Release 6.
