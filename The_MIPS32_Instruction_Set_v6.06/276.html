MFC0 IMove from Coprocessor 0 The MIPS32® Instruction Set Manual, Revision 6.06 266 Format: MFC0 rt, rd MIPS32 MFC0 rt, rd, sel MIPS32 Purpose: Move from Coprocessor 0 To move the contents of a coprocessor 0 register to a general register. Description: GPR[rt]  CPR[0,rd,sel] The contents of the coprocessor 0 register specified by the combination of rd and sel are loaded into general register rt. Not all coprocessor 0 registers support the sel field. In those instances, the sel field must be zero. Restrictions: Pre-Release 6: The results are UNDEFINED if coprocessor 0 does not contain a register as specified by rd and sel. Release 6: Reading a reserved register or a register that is not implemented for the current core configuration returns 0.
