LWC2 Load Word to Coprocessor 2 237 The MIPS32® Instruction Set Manual, Revision 6.06 Format: LWC2 rt, offset(base) MIPS32 Purpose: Load Word to Coprocessor 2 To load a word from memory to a COP2 register. Description: CPR[2,rt,0]  memory[GPR[base] + offset] The contents of the 32-bit word at the memory location specified by the aligned effective address are fetched and placed into the low word of COP2 (Coprocessor 2) general register rt. The signed offset is added to the contents of GPR base to form the effective address. Restrictions: Pre-Release 6: An Address Error exception occurs if +EffectiveAddress 1..0 ≠ 0 (not word-aligned). Release 6 allows hardware to provide address misalignment support in lieu of requiring natural alignment. Note: The pseudocode is not completely adapted for Release 6 misalignment support as the handling is implementa- tion dependent. Availability and Compatibility This instruction has been recoded for Release 6.
