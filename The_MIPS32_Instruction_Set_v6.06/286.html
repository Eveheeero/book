MOVF IMove Conditional on Floating Point False The MIPS32® Instruction Set Manual, Revision 6.06 276 Format: MOVF rd, rs, cc MIPS32, removed in Release 6 Purpose: Move Conditional on Floating Point False To test an FP condition code then conditionally move a GPR. Description: if FPConditionCode(cc) = 0 then GPR[rd]  GPR[rs] If the floating point condition code specified by CC is zero, then the contents of GPR rs are placed into GPR rd. Restrictions: Availability and Compatibility: This instruction has been removed in Release 6.
