SQRT.fmt Floating Point Square Root 382 The MIPS32® Instruction Set Manual, Revision 6.06 Format: SQRT.fmt SQRT.S fd, fs MIPS32 SQRT.D fd, fs MIPS32 Purpose: Floating Point Square Root To compute the square root of an FP value. Description: FPR[fd]  SQRT(FPR[fs]) The square root of the value in FPR fs is calculated to infinite precision, rounded according to the current rounding mode in FCSR, and placed into FPR fd. The operand and result are values in format fmt. If the value in FPR fs corresponds to – 0, the result is – 0. Restrictions: If the value in FPR fs is less than 0, an Invalid Operation condition is raised. The fields fs and fd must specify FPRs valid for operands of type fmt. If the fields are not valid, the result is UNPRE- DICTABLE. The operand must be a value in format fmt; if it is not, the result is UNPREDICTABLE and the value of the operand FPR becomes UNPREDICTABLE.
