RDPGPR Read GPR from Previous Shadow Set 335 The MIPS32® Instruction Set Manual, Revision 6.06 Format: RDPGPR rd, rt MIPS32 Release 2 Purpose: Read GPR from Previous Shadow Set To move the contents of a GPR from the previous shadow set to a current GPR. Description: GPR[rd]  SGPR[SRSCtl PSS , rt] The contents of the shadow GPR register specified by SRSCtl PSS (signifying the previous shadow set number) and rt (specifying the register number within that set) is moved to the current GPR rd. Restrictions: In implementations prior to Release 2 of the Architecture, this instruction resulted in a Reserved Instruction excep- tion.
