Vol. 3D B-11 FIELD ENCODING IN VMCS The base-address fields for ES, CS, SS, and DS in the guest-state area are defined to be natural-width (with 64 bits on processors supporting Intel 64 architecture) even though these fields are only 32-bits wide in the Intel 64 archi- tecture. VM entry ensures that the high 32 bits of these fields are cleared to 0. B.4.4 Natural-Width Host-State Fields A value of 3 in bits 11:10 of an encoding indicates a field in the host-state area. These fields are distinguished by their index value in bits 9:1. Table B-15 enumerates the natural-width host-state fields.
