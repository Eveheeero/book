Vol. 3D 38-61 INTEL® SGX INSTRUCTION REFERENCES EMODPR—Restrict the Permissions of an EPC Page Instruction Operand Encoding Description This leaf function restricts the access rights associated with an EPC page in an initialized enclave. THE RWX bits of the SECINFO parameter are treated as a permissions mask; supplying a value that does not restrict the page permissions will have no effect. This instruction can only be executed when current privilege level is 0. RBX contains the effective address of a SECINFO structure while RCX contains the effective address of an EPC page.
