17-14 Vol. 3B INTERPRETING MACHINE CHECK ERROR CODES 17.5.2 Integrated Memory Controller Machine Check Errors MC error codes associated with integrated memory controllers are reported in the IA32_MC9_STATUS− IA32_MC16_STATUS MSRs. The supported error codes follow the architectural MCACOD definition type 1MMMCCCC; see Chapter 16, “Machine-Check Architecture.” MSR_ERROR_CONTROL.[bit 1] can enable additional information logging of the IMC. The additional error informa- tion logged by the IMC is stored in IA32_MCi_STATUS and IA32_MCi_MISC, where i = 9—16. IA32_MCi_STATUS (i=9—12) logs errors from the first memory controller.
