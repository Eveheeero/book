Vol. 3C 32-5 SYSTEM MANAGEMENT MODE The following registers are saved (but not readable) and restored upon exiting SMM: • Control register CR4. (This register is cleared to all 0s when entering SMM). • The hidden segment descriptor information stored in segment registers CS, DS, ES, FS, GS, and SS. If an SMI request is issued for the purpose of powering down the processor, the values of all reserved locations in the SMM state save must be saved to nonvolatile memory.
