38-84 Vol. 3D INTEL® SGX INSTRUCTION REFERENCES EWB—Invalidate an EPC Page and Write out to Main Memory Instruction Operand Encoding Description This leaf function copies a page from the EPC to regular main memory. As part of the copying process, the page is cryptographically protected. This instruction can only be executed when current privilege level is 0.
