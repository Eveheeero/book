38-146 Vol. 3D INTEL® SGX INSTRUCTION REFERENCES EDECVIRTCHILD—Decrement VIRTCHILDCNT in SECS Instruction Operand Encoding Description This instruction decrements the SECS VIRTCHILDCNT field. This instruction can only be executed when current privilege level is 0. The content of RCX is an effective address of an EPC page. The DS segment is used to create linear address. Segment override is not supported.
