8-4 Vol. 3A TASK MANAGEMENT The processor updates dynamic fields when a task is suspended during a task switch. The following are dynamic fields: • General-purpose register fields — State of the EAX, ECX, EDX, EBX, ESP, EBP, ESI, and EDI registers prior to the task switch. • Segment selector fields — Segment selectors stored in the ES, CS, SS, DS, FS, and GS registers prior to the task switch. • EFLAGS register field — State of the EFLAGS register prior to the task switch.
