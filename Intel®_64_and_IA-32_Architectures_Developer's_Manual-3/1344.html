38-40 Vol. 3D INTEL® SGX INSTRUCTION REFERENCES EDBGWR—Write to a Debug Enclave Instruction Operand Encoding Description This leaf function copies the content in EBX/RBX to an EPC page belonging to a debug enclave. Eight bytes are written in 64-bit mode, four bytes are written in non-64-bit modes. The size of data cannot be overridden. The effective address of the target location inside the EPC is provided in the register RCX.
