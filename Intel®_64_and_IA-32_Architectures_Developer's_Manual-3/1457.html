Vol. 3D 38-153 INTEL® SGX INSTRUCTION REFERENCES ESETCONTEXT—Set the ENCLAVECONTEXT Field in SECS Instruction Operand Encoding Description The ESETCONTEXT leaf overwrites the ENCLAVECONTEXT field in the SECS. ECREATE and ELD of an SECS set the ENCLAVECONTEXT field in the SECS to the address of the SECS (for access later in ERDINFO). The ESETCONTEXT instruction allows a VMM to overwrite the default context value if necessary, for example, if the VMM is emulating ECREATE or ELD on behalf of the guest. The content of RCX is an effective address of the SECS page to be updated, RDX contains the address pointing to the value to be stored in the SECS. The DS segment is used to create linear address. Segment override is not supported. The instruction fails if: • The operand is not properly aligned. • RCX does not refer to an SECS page.
