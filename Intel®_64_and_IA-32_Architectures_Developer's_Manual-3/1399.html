Vol. 3D 38-95 INTEL® SGX INSTRUCTION REFERENCES EACCEPTCOPY—Initialize a Pending Page Instruction Operand Encoding Description This leaf function copies the contents of an existing EPC page into an uninitialized EPC page (created by EAUG). After initialization, the instruction may also modify the access rights associated with the destination EPC page. This instruction leaf can only be executed when inside the enclave. RBX contains the effective address of a SECINFO structure while RCX and RDX each contain the effective address of an EPC page.
