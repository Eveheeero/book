38-80 Vol. 3D INTEL® SGX INSTRUCTION REFERENCES ETRACKC—Activates EBLOCK Checks Instruction Operand Encoding Description The ETRACKC instruction is thread safe variant of ETRACK leaf and can be executed concurrently with other CPU threads operating on the same SECS. This leaf function provides the mechanism for hardware to track that software has completed the required TLB address clears successfully. The instruction can only be executed when the current privilege level is 0. The content of RCX is an effective address of an EPC page.
