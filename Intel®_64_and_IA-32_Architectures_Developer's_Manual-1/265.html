Vol. 1 10-17 PROGRAMMING WITH INTEL® STREAMING SIMD EXTENSIONS (INTEL® SSE) 10.5.3 Operation of FXRSTOR The FXRSTOR instruction takes a single memory operand, which is an FXSAVE area. If the value at bytes 27:24 of the FXSAVE area is not a legal value for the MXCSR register (e.g., the value sets reserved bits), execution of FXRSTOR results in a general-protection fault (#GP). Otherwise, the instruction loads x87 state and SSE state from the FXSAVE area.
