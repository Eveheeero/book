These VEX- encoded instructions generally provide the same functionality over instructions operating on XMM register that are encoded using SIMD prefixes. A subset of new functionalities listed in Table 14-4 is also extended via VEX.128 encoding. AVX provides several flexible SIMD floating-point data manipulation primitives: — permute primitives to facilitate efficient manipulation of floating-point data elements in 128-bit SIMD registers • Branch handling. AVX provides several primitives to enable handling of branches in SIMD programming: — new variable blend instructions supports four-operand syntax with non-destructive source syntax.
