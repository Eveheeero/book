TILERELEASE—Release Tile INSTRUCTION SET REFERENCE, M-U 4-708 Vol. 2B TILERELEASE—Release Tile Instruction Operand Encoding Description This instruction returns TILECFG and TILEDATA to the INIT state. Any attempt to execute the TILERELEASE instruction inside an Intel TSX transaction will result in a transaction abort. Operation zero_all_tile_data() tilecfg := 0// equivalent to 64B of zeros TILES_CONFIGURED := 0 Intel C/C++ Compiler Intrinsic Equivalent TILERELEASE void _tile_release(void); Flags Affected None. Exceptions AMX-E6; see Section 2.10, “Intel® AMX Instruction Exception Classes,” for details.
