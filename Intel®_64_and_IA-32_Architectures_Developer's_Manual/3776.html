20-68 Vol. 3B PERFORMANCE MONITORING Ice Lake microarchitecture has added a new category of Response subtype, called a Combined Response Info. To count a feature in this type, all the bits specified must be set to 1. A valid response type must be a non-zero value of the following expression: Any | ['OR' of Combined Response Info Bits | [('OR' of Supplier Info Bits) & ('OR' of Snoop Info Bits)]] If "ANY" bit[16] is set, other response type bits [17-39] are ignored.
