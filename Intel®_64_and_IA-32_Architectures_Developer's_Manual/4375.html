Vol. 3D 38-77 INTEL® SGX INSTRUCTION REFERENCES ETRACK—Activates EBLOCK Checks Instruction Operand Encoding Description This leaf function provides the mechanism for hardware to track that software has completed the required TLB address clears successfully. The instruction can only be executed when the current privilege level is 0. The content of RCX is an effective address of an EPC page.
