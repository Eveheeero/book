SETSSBSY—Mark Shadow Stack Busy INSTRUCTION SET REFERENCE, M-U 4-614 Vol. 2B SETSSBSY—Mark Shadow Stack Busy Instruction Operand Encoding Description The SETSSBSY instruction verifies the presence of a non-busy supervisor shadow stack token at the address in the IA32_PL0_SSP MSR and marks it busy. Following successful execution of the instruction, the SSP is set to the value of the IA32_PL0_SSP MSR. Operation IF (CR4.CET = 0) THEN #UD; FI; IF (IA32_S_CET.SH_STK_EN = 0) THEN #UD; FI; IF CPL > 0 THEN GP(0); FI; SSP_LA = IA32_PL0_SSP If SSP_LA not aligned to 8 bytes THEN #GP(0); FI; expected_token_value = SSP_LA (* busy bit must not be set *) new_token_value = SSP_LA | BUSY_BIT (* set busy bit; bit position 0 *) IF shadow_stack_lock_cmpxchg8B(SSP_LA, new_token_value, expected_token_value) != expected_token_value THEN #CP(SETSSBSY); FI; SSP = SSP_LA Flags Affected None.
