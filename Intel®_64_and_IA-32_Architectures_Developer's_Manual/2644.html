VRSQRTPH—Compute Reciprocals of Square Roots of Packed FP16 Values INSTRUCTION SET REFERENCE, V 5-682 Vol. 2C VRSQRTPH—Compute Reciprocals of Square Roots of Packed FP16 Values Instruction Operand Encoding Description This instruction performs a SIMD computation of the approximate reciprocals square-root of 8/16/32 packed FP16 floating-point values in the source operand (the second operand) and stores the packed FP16 floating-point results in the destination operand. The maximum relative error for this approximation is less than 2 −11 + 2 −14 . For special cases, see Table 5-38. The destination elements are updated according to the writemask.
