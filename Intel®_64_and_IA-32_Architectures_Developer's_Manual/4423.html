Vol. 3D 38-125 INTEL® SGX INSTRUCTION REFERENCES EMODPE—Extend an EPC Page Permissions Instruction Operand Encoding Description This leaf function extends the access rights associated with an existing EPC page in the running enclave. THE RWX bits of the SECINFO parameter are treated as a permissions mask; supplying a value that does not extend the page permissions will have no effect. This instruction leaf can only be executed when inside the enclave. RBX contains the effective address of a SECINFO structure while RCX contains the effective address of an EPC page.
