Vol. 3D 38-99 INTEL® SGX INSTRUCTION REFERENCES EDECCSSA—Decrements TCS.CSSA Instruction Operand Encoding Description This leaf function changes the current SSA frame by decrementing TCS.CSSA for the current enclave thread. If the enclave has enabled CET shadow stacks or indirect branch tracking, then EDECCSSA also changes the current CET state save frame. This instruction leaf can only be executed inside an enclave.
