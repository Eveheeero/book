BNDMOV—Move Bounds INSTRUCTION SET REFERENCE, A-L Vol. 2A 3-117 BNDMOV—Move Bounds Instruction Operand Encoding Description BNDMOV moves a pair of lower and upper bound values from the source operand (the second operand) to the destination (the first operand). Each operation is 128-bit move. The exceptions are same as the MOV instruction. The memory format for loading/store bounds in 64-bit mode is shown in Figure 3-5. This instruction does not change flags.
