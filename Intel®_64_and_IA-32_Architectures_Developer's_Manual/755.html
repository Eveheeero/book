CLD—Clear Direction Flag INSTRUCTION SET REFERENCE, A-L Vol. 2A 3-159 CLD—Clear Direction Flag Instruction Operand Encoding Description Clears the DF flag in the EFLAGS register. When the DF flag is set to 0, string operations increment the index regis- ters (ESI and/or EDI). Operation is the same in all modes. Operation DF := 0; Flags Affected The DF flag is set to 0. The CF, OF, ZF, SF, AF, and PF flags are unaffected. Exceptions (All Operating Modes) #UD If the LOCK prefix is used.
