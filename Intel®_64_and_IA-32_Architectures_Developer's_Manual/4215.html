TMA Packet Definition Name TSC/MTC Alignment (TMA) Packet Packet Format Dependencies IA32_RTIT_CTL.MTCEn && IA32_RTIT_CTL.TSCEn && TriggerEn Generation Sce- nario Sent with any TSC packet. Description The TMA packet serves to provide the information needed to allow the decoder to correlate MTC packets with TSC packets. With this packet, when a MTC packet is encountered, the decoder can determine how many timestamp counter ticks have passed since the last TSC or MTC packet. See Section 33.8.3.2 for details on how to make this cal- culation. Application TMA is always sent immediately following a TSC packet, and the payload values are consistent with the TSC payload value.
