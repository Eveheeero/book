POR—Bitwise Logical OR INSTRUCTION SET REFERENCE, M-U Vol. 2B 4-405 POR—Bitwise Logical OR Instruction Operand Encoding Description Performs a bitwise logical OR operation on the source operand (second operand) and the destination operand (first operand) and stores the result in the destination operand. Each bit of the result is set to 1 if either or both of the corresponding bits of the first and second operands are 1; otherwise, it is set to 0. In 64-bit mode and not encoded with VEX/EVEX, using a REX prefix in the form of REX.R permits this instruction to access additional registers (XMM8-XMM15).
