VCMPPH—Compare Packed FP16 Values INSTRUCTION SET REFERENCE, V Vol. 2C 5-21 VCMPPH—Compare Packed FP16 Values Instruction Operand Encoding Description This instruction compares packed FP16 values from source operands and stores the result in the destination mask operand. The comparison predicate operand (immediate byte bits 4:0) specifies the type of comparison performed on each of the pairs of packed values. The destination elements are updated according to the writemask.
