Hence, a PEBS record's size and layout vary based on the selected groups. The MSR also allows software to select LBR depth for branch data records. 0 Memory Info. Setting this bit will capture memory information such as the linear address, data source and latency of the memory access in the PEBS record. 1 GPRs. Setting this bit will capture the contents of the General Purpose registers in the PEBS record. 2 XMMs. Setting this bit will capture the contents of the XMM registers in the PEBS record. 3 LBRs. Setting this bit will capture LBR TO, FROM, and INFO in the PEBS record. 23:4 Reserved. 31:24 LBR Entries. Set the field to the desired number of entries - 1.
