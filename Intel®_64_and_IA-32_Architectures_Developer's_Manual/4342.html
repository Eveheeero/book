38-44 Vol. 3D INTEL® SGX INSTRUCTION REFERENCES EEXTEND—Extend Uninitialized Enclave Measurement by 256 Bytes Instruction Operand Encoding Description This leaf function updates the MRENCLAVE measurement register of an SECS with the measurement of an EXTEND string compromising of “EEXTEND” || ENCLAVEOFFSET || PADDING || 256 bytes of the enclave page. This instruc- tion can only be executed when current privilege level is 0 and the enclave is uninitialized. RBX contains the effective address of the SECS of the region to be measured. The address must be the same as the one used to add the page into the enclave. RCX contains the effective address of the 256 byte region of an EPC page to be measured. The DS segment is used to create linear addresses. Segment override is not supported.
