VFCMADDCPH/VFMADDCPH—Complex Multiply and Accumulate FP16 Values INSTRUCTION SET REFERENCE, V 5-170 Vol. 2C VFCMADDCPH/VFMADDCPH—Complex Multiply and Accumulate FP16 Values Instruction Operand Encoding Description This instruction performs a complex multiply and accumulate operation. There are normal and complex conjugate forms of the operation. The broadcasting and masking for this operation is done on 32-bit quantities representing a pair of FP16 values. Rounding is performed at every FMA (fused multiply and add) boundary. Execution occurs as if all MXCSR excep- tions are masked. MXCSR status bits are updated to reflect exceptional conditions.
