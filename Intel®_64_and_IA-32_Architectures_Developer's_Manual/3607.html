Vol. 3B 17-25 INTERPRETING MACHINE CHECK ERROR CODES 17.9.3 Integrated Memory Controller Machine Check Errors MC error codes associated with integrated memory controllers are reported in the IA32_MC13_STATUS—IA32_MC18_STATUS MSRs. The supported error codes follow the architectural MCACOD definition type 1MMMCCCC; see Chapter 16, “Machine-Check Architecture.” IA32_MCi_STATUS (i=13,14,17) logs errors from the first memory controller.
