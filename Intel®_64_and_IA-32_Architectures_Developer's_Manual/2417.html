VPDPBUSD—Multiply and Add Unsigned and Signed Bytes INSTRUCTION SET REFERENCE, V Vol. 2C 5-455 VPDPBUSD—Multiply and Add Unsigned and Signed Bytes Instruction Operand Encoding Description Multiplies the individual unsigned bytes of the first source operand by the corresponding signed bytes of the second source operand, producing intermediate signed word results. The word results are then summed and accumulated in the destination dword element size operand. This instruction supports memory fault suppression.
