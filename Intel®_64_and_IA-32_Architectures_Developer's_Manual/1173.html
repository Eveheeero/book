KUNPCKBW/KUNPCKWD/KUNPCKDQ—Unpack for Mask Registers INSTRUCTION SET REFERENCE, A-L Vol. 2A 3-577 KUNPCKBW/KUNPCKWD/KUNPCKDQ—Unpack for Mask Registers Instruction Operand Encoding Description Unpacks the lower 8/16/32 bits of the second and third operands (source operands) into the low part of the first operand (destination operand), starting from the low bytes. The result is zero-extended in the destination. Operation KUNPCKBW DEST[7:0] := SRC2[7:0] DEST[15:8] := SRC1[7:0] DEST[MAX_KL-1:16] := 0 KUNPCKWD DEST[15:0] := SRC2[15:0] DEST[31:16] := SRC1[15:0] DEST[MAX_KL-1:32] := 0 KUNPCKDQ DEST[31:0] := SRC2[31:0] DEST[63:32] := SRC1[31:0] DEST[MAX_KL-1:64] := 0 Intel C/C++ Compiler Intrinsic Equivalent KUNPCKBW __mmask16 _mm512_kunpackb(__mmask16 a, __mmask16 b); KUNPCKDQ __mmask64 _mm512_kunpackd(__mmask64 a, __mmask64 b); KUNPCKWD __mmask32 _mm512_kunpackw(__mmask32 a, __mmask32 b); Flags Affected None. SIMD Floating-Point Exceptions None.
