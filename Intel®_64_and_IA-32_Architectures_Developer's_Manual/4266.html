35-12 Vol. 3D ENCLAVE ACCESS CONTROL AND DATA STRUCTURES 35.9.2.2 Page Fault Error Code Table 35-14 contains page fault error code that may be reported in EXINFO.ERRCD. 35.10 CET STATE SAVE AREA FRAME The CET state save area consists of an array of CET state save frames. The number of CET state save frames is equal to the TCS.NSSA. The current CET SSA frame is indicated by TCS.CSSA. The offset of the CET state save area is specified by TCS.OCETSSA. 35.11 PAGE INFORMATION (PAGEINFO) PAGEINFO is an architectural data structure that is used as a parameter to the EPC-management instructions.
