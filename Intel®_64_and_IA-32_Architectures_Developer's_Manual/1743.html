PXOR—Logical Exclusive OR INSTRUCTION SET REFERENCE, M-U Vol. 2B 4-523 PXOR—Logical Exclusive OR Instruction Operand Encoding Description Performs a bitwise logical exclusive-OR (XOR) operation on the source operand (second operand) and the destina- tion operand (first operand) and stores the result in the destination operand. Each bit of the result is 1 if the corre- sponding bits of the two operands are different; each bit is 0 if the corresponding bits of the operands are the same. In 64-bit mode and not encoded with VEX/EVEX, using a REX prefix in the form of REX.R permits this instruction to access additional registers (XMM8-XMM15). Legacy SSE instructions 64-bit operand: The source operand can be an MMX technology register or a 64-bit memory location. The destination operand is an MMX technology register.
