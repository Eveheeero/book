35-4 Vol. 3D ENCLAVE ACCESS CONTROL AND DATA STRUCTURES 35.6 INTEL® SGX DATA STRUCTURES OVERVIEW Enclave operation is managed via a collection of data structures. Many of the top-level data structures contain sub- structures. The top-level data structures relate to parameters that may be used in enclave setup/maintenance, by Intel SGX instructions, or AEX event.
