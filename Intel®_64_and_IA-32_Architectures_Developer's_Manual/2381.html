VPBROADCASTB/W/D/Q—Load With Broadcast Integer Data From General Purpose Register INSTRUCTION SET REFERENCE, V Vol. 2C 5-419 VPBROADCASTB/W/D/Q—Load With Broadcast Integer Data From General Purpose Register Instruction Operand Encoding Description Broadcasts a 8-bit, 16-bit, 32-bit or 64-bit value from a general-purpose register (the second operand) to all the locations in the destination vector register (the first operand) using the writemask k1. EVEX.vvvv is reserved and must be 1111b otherwise instructions will #UD.
