KORW/KORB/KORQ/KORD—Bitwise Logical OR Masks INSTRUCTION SET REFERENCE, A-L 3-568 Vol. 2A KORW/KORB/KORQ/KORD—Bitwise Logical OR Masks Instruction Operand Encoding Description Performs a bitwise OR between the vector mask k2 and the vector mask k3, and writes the result into vector mask k1 (three-operand form). Operation KORW DEST[15:0] := SRC1[15:0] BITWISE OR SRC2[15:0] DEST[MAX_KL-1:16] := 0 KORB DEST[7:0] := SRC1[7:0] BITWISE OR SRC2[7:0] DEST[MAX_KL-1:8] := 0 KORQ DEST[63:0] := SRC1[63:0] BITWISE OR SRC2[63:0] DEST[MAX_KL-1:64] := 0 KORD DEST[31:0] := SRC1[31:0] BITWISE OR SRC2[31:0] DEST[MAX_KL-1:32] := 0 Intel C/C++ Compiler Intrinsic Equivalent KORW __mmask16 _mm512_kor(__mmask16 a, __mmask16 b); Flags Affected None. SIMD Floating-Point Exceptions None.
