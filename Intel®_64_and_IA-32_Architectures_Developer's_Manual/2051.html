VCVTSI2SH—Convert a Signed Doubleword/Quadword Integer to an FP16 Value INSTRUCTION SET REFERENCE, V Vol. 2C 5-89 VCVTSI2SH—Convert a Signed Doubleword/Quadword Integer to an FP16 Value Instruction Operand Encoding Description This instruction converts a signed doubleword integer (or signed quadword integer if operand size is 64 bits) in the second source operand to an FP16 value in the destination operand. The result is stored in the low word of the desti- nation operand. When conversion is inexact, the value returned is rounded according to the rounding control bits in the MXCSR register or embedded rounding controls. The second source operand can be a general-purpose register or a 32/64-bit memory location. The first source and destination operands are XMM registers. Bits 127:16 of the XMM register destination are copied from corre- sponding bits in the first source operand. Bits MAXVL-1:128 of the destination register are zeroed. If the result of the convert operation is overflow and MXCSR.OM=0 then a SIMD exception will be raised with OE=1, PE=1. Operation VCVTSI2SH dest, src1, src2 IF *SRC2 is a register* and (EVEX.b = 1): SET_RM(EVEX.RC) ELSE: SET_RM(MXCSR.RC) IF 64-mode and OperandSize == 64: DEST.fp16[0] := Convert_integer64_to_fp16(SRC2.qword) ELSE: DEST.fp16[0] := Convert_integer32_to_fp16(SRC2.dword) DEST[127:16] := SRC1[127:16] DEST[MAXVL-1:128] := 0 Intel C/C++ Compiler Intrinsic Equivalent VCVTSI2SH __m128h _mm_cvt_roundi32_sh (__m128h a, int b, int rounding); VCVTSI2SH __m128h _mm_cvt_roundi64_sh (__m128h a, __int64 b, int rounding); VCVTSI2SH __m128h _mm_cvti32_sh (__m128h a, int b); VCVTSI2SH __m128h _mm_cvti64_sh (__m128h a, __int64 b); Opcode/ Instruction Op/ En 64/32 bit Mode Support CPUID Feature Flag Description EVEX.LLIG.F3.MAP5.W0 2A /r VCVTSI2SH xmm1, xmm2, r32/m32 {er} A V/V 1 NOTES: 1.
