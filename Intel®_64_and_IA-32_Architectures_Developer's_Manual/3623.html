Vol. 3B 17-41 INTERPRETING MACHINE CHECK ERROR CODES 17.13.3 Integrated Memory Controller Machine Check Errors MC error codes associated with integrated memory controllers for the 4th generation Intel ® Xeon ® Scalable Processor Family based on Sapphire Rapids microarchitecture are reported in the IA32_MC13_STATUS− IA32_MC20_STATUS MSRs. The supported error codes follow the architectural MCACOD definition type 1MMMCCCC; see Chapter 16, “Machine-Check Architecture.” Table 17-42.
