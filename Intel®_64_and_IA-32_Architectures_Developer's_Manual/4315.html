Vol. 3D 38-17 INTEL® SGX INSTRUCTION REFERENCES EADD—Add a Page to an Uninitialized Enclave Instruction Operand Encoding Description This leaf function copies a source page from non-enclave memory into the EPC, associates the EPC page with an SECS page residing in the EPC, and stores the linear address and security attributes in EPCM. As part of the asso- ciation, the enclave offset and the security attributes are measured and extended into the SECS.MRENCLAVE. This instruction can only be executed when current privilege level is 0. RBX contains the effective address of a PAGEINFO structure while RCX contains the effective address of an EPC page.
