KANDW/KANDB/KANDQ/KANDD—Bitwise Logical AND Masks INSTRUCTION SET REFERENCE, A-L Vol. 2A 3-563 KANDW/KANDB/KANDQ/KANDD—Bitwise Logical AND Masks Instruction Operand Encoding Description Performs a bitwise AND between the vector mask k2 and the vector mask k3, and writes the result into vector mask k1.
