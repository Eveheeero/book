Vol. 3B 20-79 PERFORMANCE MONITORING 20.4.1.1.3 Average Offcore Request Latency Measurement Measurement of average latency of offcore transaction requests can be enabled using MSR_OFFCORE_RSP0.[bit 38] with the choice of request type specified in MSR_OFFCORE_RSP0.[bit 15:0]. Refer to Section 20.5.2.3, “Average Offcore Request Latency Measurement,” for typical usage. Note that MSR_OFFCORE_RESPx registers are not shared between cores in Knights Landing.
