GETSEC[PARAMETERS]—Report the SMX Parameters SAFER MODE EXTENSIONS REFERENCE Vol. 2D 7-33 GETSEC[PARAMETERS]—Report the SMX Parameters Description The GETSEC[PARAMETERS] instruction returns specific parameter information for SMX features supported by the processor. Parameter information is returned in EAX, EBX, and ECX, with the input parameter selected using EBX. Software retrieves parameter information by searching with an input index for EBX starting at 0, and then reading the returned results in EAX, EBX, and ECX. EAX[4:0] is designated to return a parameter type field indicating if a parameter is available and what type it is. If EAX[4:0] is returned with 0, this designates a null parameter and indi- cates no more parameters are available.
