VCVTUSI2SD—Convert Unsigned Integer to Scalar Double Precision Floating-Point Value INSTRUCTION SET REFERENCE, V 5-136 Vol. 2C VCVTUSI2SD—Convert Unsigned Integer to Scalar Double Precision Floating-Point Value Instruction Operand Encoding Description Converts an unsigned doubleword integer (or unsigned quadword integer if operand size is 64 bits) in the second source operand to a double precision floating-point value in the destination operand. The result is stored in the low quadword of the destination operand. When conversion is inexact, the value returned is rounded according to the rounding control bits in the MXCSR register. The second source operand can be a general-purpose register or a 32/64-bit memory location. The first source and destination operands are XMM registers. Bits (127:64) of the XMM register destination are copied from corre- sponding bits in the first source operand. Bits (MAXVL-1:128) of the destination register are zeroed. EVEX.W1 version: promotes the instruction to use 64-bit input value in 64-bit mode. EVEX.W0 version: attempt to encode this instruction with EVEX embedded rounding is ignored.
