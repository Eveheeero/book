TGEU Trap if Greater or Equal Unsigned 355 MIPS® Architecture for Programmers Volume II-B: microMIPS32™ Instruction Set, Revision 6.05 Format: TGEU rs, rt microMIPS Purpose: Trap if Greater or Equal Unsigned To compare GPRs and do a conditional trap. Description: if GPR[rs]  GPR[rt] then Trap Compare the contents of GPR rs and GPR rt as unsigned integers. If GPR rs is greater than or equal to GPR rt, then take a Trap exception. The contents of the code field are ignored by hardware and may be used to encode information for system software. To retrieve the information, the system software may load the instruction word from memory. Alternatively, if CP0 BadInstr is implemented, the code field may be obtained from BadInstr .
