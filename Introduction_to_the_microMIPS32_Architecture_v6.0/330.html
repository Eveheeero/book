SLTU ISet on Less Than Unsigned MIPS® Architecture for Programmers Volume II-B: microMIPS32™ Instruction Set, Revision 6.05 330 Format: SLTU rd, rs, rt microMIPS Purpose: Set on Less Than Unsigned To record the result of an unsigned less-than comparison. Description: GPR[rd]  (GPR[rs] < GPR[rt]) Compare the contents of GPR rs and GPR rt as unsigned integers; record the Boolean result of the comparison in GPR rd. If GPR rs is less than GPR rt, the result is 1 (true); otherwise, it is 0 (false). The arithmetic comparison does not cause an Integer Overflow exception.
