AUIPC Add Upper Immediate to PC MIPS® Architecture for Programmers Volume II-B: microMIPS32™ Instruction Set, Revision 6.05 127 Format: AUIPC rt, immediate microMIPS32 Release 6 Purpose: Add Upper Immediate to PC Description: GPR[rt]  ( PC + ( immediate << 16 ) ) This instruction performs a PC-relative address calculation. The 16-bit immediate is shifted left by 16 bits, sign- extended, and added to the address of the AUIPC instruction. The result is placed in GPR rt. Restrictions: None Availability and Compatibility: This instruction is introduced by and required as of Release 6.
