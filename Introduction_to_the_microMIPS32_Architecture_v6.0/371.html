TNE Trap if Not Equal 371 MIPS® Architecture for Programmers Volume II-B: microMIPS32™ Instruction Set, Revision 6.05 Format: TNE rs, rt microMIPS Purpose: Trap if Not Equal To compare GPRs and do a conditional trap. Description: if GPR[rs] ≠ GPR[rt] then Trap Compare the contents of GPR rs and GPR rt as signed integers. If GPR rs is not equal to GPR rt, then take a Trap exception. The contents of the code field are ignored by hardware and may be used to encode information for system software. To retrieve the information, system software must load the instruction word from memory. Alternatively, if CP0 BadInstr is implemented, the code field may be obtained from BadInstr .
