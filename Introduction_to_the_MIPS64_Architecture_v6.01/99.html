The Floating Point Enables Register (FENR) is an alternative way to read and write the Enables, FS, and RM fields that also appear in FCSR. Figure 6.17 shows the format of the FENR register; Table 6.13 describes the FENR regis- ter fields. The value held in a floating point register (FPR) is either uninterpreted, or one of the valid numeric formats: single, double, paired-single floating-point, word and long fixed point.
