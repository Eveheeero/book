The Release 6 ISA defines the BITSWAP instruction, which reverses the bits in every byte of its operand. MIPS 64 Release 6 similarly defines DBITSWAP. Release 6 BITSWAP corresponds to MIPS DSP Module BITREV. The Release 6 ISA defines the ALIGN instruction that concatenates its two operands and the same-width contiguous subset from the concatenation at byte granularity. ALIGN is useful for extracted data at a misaligned memory address from two aligned memory load results. MIPS 64 Release 6 similarly defines DALIGN.
