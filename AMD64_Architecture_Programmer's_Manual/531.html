Segmentation Data Structures Figure 4-2 on page 77 shows the registers used by the segmentation mechanism. The registers have the following relationship to the data structures: • Segment Registers—The six segment registers (CS, DS, ES, FS, GS, and SS) are used to point to the user segments. A segment selector selects a descriptor when it is loaded into one of the segment registers. This causes the processor to automatically load the selected descriptor into a software- invisible portion of the segment register. • Descriptor-Table Registers—The three descriptor-table registers (GDTR, LDTR, and IDTR) are used to point to the system segments.
