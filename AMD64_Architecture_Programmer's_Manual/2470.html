Instruction Reference PUNPCKHWD, VPUNPCKHWD 509 26568—Rev. 3.25—November 2021 AMD64 Technology YMM Encoding The first source operand is a YMM register. The second source operand is either a YMM register or a 256-bit memory location. The destination is a third YMM register.
