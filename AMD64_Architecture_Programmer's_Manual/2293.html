332 PEXTRQ, VPEXTRQ Instruction Reference AMD64 Technology 26568—Rev. 3.25—November 2021 Extracts a quadword from a source register and writes it to an 64-bit memory location or to a 64-bit general-purpose register. Bit [0] of an immediate byte operand selects the quadword to be extracted: There are legacy and extended forms of the instruction: PEXTRQ The encoding is the same as PEXTRD, with REX.W = 1. The source operand is an XMM register and the destination is either an 64-bit memory location or a 64-bit general-purpose register. VPEXTRQ The extended form of the instruction has a 128-bit encoding only. The encoding is the same as VPEXTRD, with VEX.W = 1. The source operand is an XMM register and the destination is either an 64-bit memory location or a 64-bit general-purpose register.
