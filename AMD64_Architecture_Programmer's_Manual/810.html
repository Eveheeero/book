Sub-functions (ECX > 0) provide details concerning features and support of processor states enumerated in the root function. 11.5.2 XFEATURE_ENABLED_MASK XFEATURE_ENABLED_MASK is set up by privileged software to enable the saving and restoring of extended processor architectural state information supported by a specific processor. Clearing defined bit fields in this mask inhibits the XSAVE instruction from saving (and XRSTOR from restoring) this state information. XFEATURE_ENABLED_MASK is addressed as XCR0 in the extended control register space and is accessed via the XSETBV and XGETBV instructions. XFEATURE_ENABLED_MASK is defined as follows: Figure 11-8.
