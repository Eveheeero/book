64-Bit Media PFRCPIT1 119 Instruction Reference 26569—Rev. 3.16—November 2021 AMD64 Technology Operation mmx1[31:0] = Compress (2 - mmx1[31:0] * (mmx2/mem64[31:0]) - 2 31 ); mmx1[63:32] = Compress (2 - mmx1[63:32] * (mmx2/mem64[63:32]) - 2 31 ); where: “Compress” means discard the 8 redundant most-significant bits after the hidden integer bit. Examples The general Newton-Raphson recurrence for the reciprocal 1/b is: Z i +1 ← Z i • (2 – b • Z i ) The following code sequence computes a 24-bit approximation to a/b with one Newton-Raphson iteration: X 0 = PFRCP(b) X 1 = PFRCPIT1(b, X 0 ) X 2 = PFRCPIT2(X 1 , X 0 ) q = PFMUL(a, X 2 ) a/b is formed in the last step by multiplying the reciprocal approximation by a.
