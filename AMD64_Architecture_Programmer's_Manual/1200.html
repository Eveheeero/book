745 SVM Intercept Exit Codes AMD64 Technology 24593—Rev. 3.41—June 2023 Appendix C SVM Intercept Exit Codes When the VMRUN instruction exits (back to the host), an exit/reason code is stored in the EXIT- CODE field in the VMCB.
