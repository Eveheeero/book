General-Purpose 81 Instruction Reference 24594—Rev. 3.35—June 2023 AMD64 Technology Adds the value in a register (first operand) with a register or memory (second operand) and the carry flag, and stores the result in the first operand location. This instruction sets the CF based on the unsigned addition. This instruction is useful in multi-precision addition algorithms. This is an ADX instructions.
