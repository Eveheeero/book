YMM Move Operations Streaming-store versions of the move instructions (also known as non-temporal moves) bypass the cache when storing data that is accessed only once. This maximizes memory-bus utilization and minimizes cache pollution. The move-mask instruction stores specific bytes from one vector, as selected by mask values in a second vector.
