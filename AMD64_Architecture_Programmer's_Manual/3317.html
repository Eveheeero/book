x87 Floating-Point FUCOMIx 319 Instruction Reference 26569—Rev. 3.16—November 2021 AMD64 Technology Compares the contents of ST(0) with the contents of another floating-point register, and sets the zero flag (ZF), parity flag (PF), and carry flag (CF) as shown in the rFLAGS Affected table below. Unlike FCOMI and FCOMIP, the FUCOMI and FUCOMIP instructions do not set the invalid- operation exception (IE) bit in the x87 status word for QNaNs. After completing the comparison, FUCOMIP pops the x87 register stack. Support for the FCOMI(P) instruction can be determined by executing either CPUID function 0000_0001h or CPUID function 8000_0001. Support is indicated when both the EDX[FPU] (bit 0) and EDX[CMOV] (bit 15) feature flags are set.
