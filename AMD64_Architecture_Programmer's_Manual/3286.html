288 FRNDINT x87 Floating-Point Instruction Reference AMD64 Technology 26569—Rev. 3.16—November 2021 Rounds the value in ST(0) to an integer, depending on the setting of the rounding control (RC) field of the x87 control word, and stores the result in ST(0). If the initial value in ST(0) is ∞, the instruction does not change ST(0). If the value in ST(0) is not an integer, it sets the precision exception (PE) bit of the x87 status word to 1.
