286 FPTAN x87 Floating-Point Instruction Reference AMD64 Technology 26569—Rev. 3.16—November 2021 Computes the tangent of the radian value in ST(0), stores the result in ST(0), and pushes a value of 1.0 onto the x87 register stack. The source value must be between –2 63 and +2 63 radians. If the source value lies outside the specified range, the instruction sets the C2 bit of the x87 status word to 1 and does not change the value in ST(0).
