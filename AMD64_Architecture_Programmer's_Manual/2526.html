Instruction Reference SQRTPD, VSQRTPD 565 26568—Rev. 3.25—November 2021 AMD64 Technology Computes the square root of each packed double-precision floating-point value in a source operand and writes the result to the corresponding quadword of the destination. Performing the square root of +infinity returns +infinity. There are legacy and extended forms of the instruction: SQRTPD Computes two values. The destination is an XMM register. The source operand is either an XMM register or a 128-bit memory location. Bits [255:128] of the YMM register that corresponds to the destination are not affected. VSQRTPD The extended form of the instruction has both 128-bit and 256-bit encodings: XMM Encoding Computes two values. The source operand is either an XMM register or a 128-bit memory location. The destination is an XMM register. Bits [255:128] of the YMM register that corresponds to the des- tination are cleared. YMM Encoding Computes four values. The source operand is either a YMM register or a 256-bit memory location. The destination is a YMM register.
