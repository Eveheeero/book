604 VCVTPH2PS Instruction Reference AMD64 Technology 26568—Rev. 3.25—November 2021 YMM Encoding Converts eight packed 16-bit floating-point values in the low-order 128 bits of a YMM register or in a 128-bit memory location to eight packed single-precision floating-point values and writes the con- verted values to a YMM destination register.
