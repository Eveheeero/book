252 FILD x87 Floating-Point Instruction Reference AMD64 Technology 26569—Rev. 3.16—November 2021 Converts a signed-integer in memory to double-extended-precision format and pushes the value onto the x87 register stack. The value can be a 16-bit, 32-bit, or 64- bit integer value. Signed values from memory can always be represented exactly in x87 registers without rounding. Related Instructions FLD, FST, FSTP, FIST, FISTP, FBLD, FBSTP rFLAGS Affected None x87 Condition Code FILD Floating-Point Load Integer Mnemonic Opcode Description FILD mem16int DF /0 Push the contents of mem16int onto the x87 register stack. FILD mem32int DB /0 Push the contents of mem32int onto the x87 register stack. FILD mem64int DF /5 Push the contents of mem64int onto the x87 register stack. x87 Condition Code Value Description C0 U C1 0 No stack overflow. 1 x87 stack overflow, if an x87 register stack fault was detected. C2 U C3 U Note: A flag set to 1 or cleared to 0 is M (modified). Unaffected flags are blank. Undefined flags are U. [AMD Public Use]
