E.4.5 Function 8000_0006h—L2 Cache and TLB and L3 Cache Information This function provides the second level cache and TLB characteristics for the logical processor that executes the instruction. The EDX register returns the processor’s third level cache characteristics that are shared by all logical processors in the package. The value returned in EAX provides information about the L2 TLB for 2-MB and 4-MB pages.
