Instruction Reference PSHUFD, VPSHUFD 439 26568—Rev. 3.25—November 2021 AMD64 Technology There are legacy and extended forms of the instruction: PSHUFD The source operand is either an XMM register or a 128-bit memory location. The destination is an XMM register. Bits [255:128] of the YMM register that corresponds to the destination are not affected. VPSHUFD The extended form of the instruction has 128-bit and 256-bit encodings. XMM Encoding The source operand is either an XMM register or a 128-bit memory location. The destination is an XMM register. Bits [255:128] of the YMM register that corresponds to the destination are cleared. YMM Encoding The source operand is either a YMM register or a 256-bit memory location. The destination is a YMM register.
