312 PCMPESTRM, VPCMPESTRM Instruction Reference AMD64 Technology 26568—Rev. 3.25—November 2021 The first source operand is an XMM register. The second source operand is either an XMM register or a 128-bit memory location. The mask result is written to the XMM0 register. Bits [255:128] of the YMM0 register are cleared.
