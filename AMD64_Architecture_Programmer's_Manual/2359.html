398 PMOVSXBW, VPMOVSXBW Instruction Reference AMD64 Technology 26568—Rev. 3.25—November 2021 Sign-extends eight or sixteen packed 8-bit signed integers in the source operand to 16 bits and writes the packed word signed integers to the destination. If the source operand is a register, the eight 8-bit signed integers are taken from the lower half of the register. There are legacy and extended forms of the instruction: PMOVSXBW The source operand is either an XMM register or a 64-bit memory location. The destination is an XMM register. Bits [255:128] of the YMM register that corresponds to the destination are not affected. VPMOVSXBW The extended form of the instruction has 128-bit and 256-bit encodings. XMM Encoding The source operand is either an XMM register or a 64-bit memory location. The destination is an XMM register. Bits [255:128] of the YMM register that corresponds to the destination are cleared. YMM Encoding The source operand is either an XMM register or a 128-bit memory location. The destination is a YMM register. Instruction Support For more on using the CPUID instruction to obtain processor feature support information, see Appen- dix E of Volume 3.
