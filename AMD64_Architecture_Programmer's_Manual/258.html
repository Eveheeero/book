The priority of the exceptions is described in “SIMD Floating-Point Exception Priority” on page 224. 4.10.2.3 Invalid-Operation Exception (IE) The IE exception occurs due to one of the attempted invalid operations shown in Table 4-13. 4.10.2.4 Denormalized-Operand Exception (DE) The DE exception occurs when one of the source operands of an instruction is in denormalized form, as described in “Denormalized (Tiny) Numbers” on page 125. 4.10.2.5 Zero-Divide Exception (ZE) The ZE exception occurs when an instruction attempts to divide zero into a non-zero finite dividend.
