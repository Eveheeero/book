(V)MASKMOVDQU Move Mask Operation 4.6.1.3 Move Mask • (V)PMOVMSKB—Packed Move Mask Byte The (V)PMOVMSKB instruction moves the most-significant bit of each byte in an XMM register to the low-order word of a 32-bit or 64-bit general-purpose register, with zero-extension. The instruction is useful for extracting bits from mask patterns, or zero values from quantized data, or sign bits— resulting in a byte that can be used for data-dependent branching.
