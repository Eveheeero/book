Or • POR—Packed Logical Bitwise OR The POR instruction performs a bitwise logical OR of the values in the first and second operands and writes the result to the destination. Exclusive Or • PXOR—Packed Logical Bitwise Exclusive OR The PXOR instruction performs a bitwise logical exclusive OR of the values in the first and second operands and writes the result to the destination. PXOR can be used to clear all bits in an MMX register by specifying the same register for both operands. PXOR can also used with the value 8000000080000000h to change the sign bits of the elements of a 64-bit media floating-point vector operand. This method is equivalent to the x87 floating-point change sign (FCHS) instruction. 5.6.10 Save and Restore State These instructions save and restore the processor state for 64-bit media instructions.
