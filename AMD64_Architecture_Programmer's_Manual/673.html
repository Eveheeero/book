Memory System 218 24593—Rev. 3.41—June 2023 AMD64 Technology Figure 7-7. MTRR Mapping of Physical Memory MTRRs are 64-bit model-specific registers (MSRs). They are read using the RDMSR instruction and written using the WRMSR instruction. See “Memory-Typing MSRs” on page 720 for a listing of the MTRR MSR numbers. The following sections describe the types of MTRRs and their function. Fixed-Range MTRRs. The fixed-range MTRRs are used to characterize the first 1 Mbyte of physical memory. Each fixed-range MTRR contains eight type fields for characterizing a total of eight memory ranges. Fixed-range MTRRs support extended type-field encodings as described in “Extended Fixed- Range MTRR Type-Field Encodings” on page 231. The extended type field allows a fixed-range MTRR to be used as a fixed-range IORR.
