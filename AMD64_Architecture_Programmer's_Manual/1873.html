606 Obtaining Processor Information Via the CPUID Instruction AMD64 Technology 24594—Rev. 3.35—June 2023 E.3.7 Functions 8h–Ah—Reserved E.3.8 Function Bh — Extended Topology Enumeration CPUID Fn0000_000B enumerates each level in the processor’s topological hierarchy. The hierarchy level is specified by the input value passed in the ECX register. If this function is executed with an unimplemented level (passed in ECX), the instruction returns all zeros in the EAX register.
