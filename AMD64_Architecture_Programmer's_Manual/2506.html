Instruction Reference RSQRTSS, VRSQRTSS 545 26568—Rev. 3.25—November 2021 AMD64 Technology Execute 4 rounds of a SHA1 operation using the 4 double words (A, B, C, D) from the first source operand, and value E from the second operand. The lower two bits of the immediate are used to spec- ify the function and constant appropriate for the current round of processing. The resulting (A, B, C, D) is placed in the destination register which is the same as the first source register.
