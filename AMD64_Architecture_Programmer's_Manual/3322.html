324 FXCH x87 Floating-Point Instruction Reference AMD64 Technology 26569—Rev. 3.16—November 2021 Exchanges the value in ST(0) with the value in any other x87 register. If no operand is specified, the instruction exchanges the values in ST(0) and ST(1). Use this instruction to move a value from an x87 register to ST(0) for subsequent processing by a floating-point instruction that can only operate on ST(0).
