Instruction Reference VPPERM 819 26568—Rev. 3.25—November 2021 AMD64 Technology Selects 16 of 32 packed bytes from two concatenated sources, applies a logical transformation to each selected byte, then writes the byte to a specified position in the destination. There are four operands: VPPERM dest, src1, src2, src3 The second (src2) and first (src1) sources are concatenated to form the 32-byte source. The src1 operand is an XMM register specified by XOP.vvvv. The third source (src3) contains 16 control bytes. Each control byte specifies the source byte and the logical operation to perform on that byte. The order of the bytes in the destination is the same as that of the control bytes in the src3. For each byte of the 16-byte result, the corresponding src3 byte is used as follows: • Bits [7:5] select a logical operation to perform on the selected byte. • Bits [4:0] select a source byte to move from src2:src1. XOP.W and an immediate byte (imm8) determine register configuration. • When XOP.W = 0, src2 is either an XMM register or a 128-bit memory location specified by ModRM.r/m and src3 is an XMM register specified by imm8[7:4].
