76 General-Purpose Instruction Reference AMD64 Technology 24594—Rev. 3.35—June 2023 Converts two unpacked BCD digits in the AL (least significant) and AH (most significant) registers to a single binary value in the AL register. The instruction is coded without explicit operands: AAD The instruction performs the following operation on the contents of AL and AH using the formula: AL dAHAL After the conversion, AH is cleared to 00h. In most modern assemblers, the AAD instruction adjusts from base-10 values. However, by coding the instruction directly in binary, it can adjust from any base specified by the immediate byte value (ib) suffixed onto the D5h opcode. For example, code D508h for octal, D50Ah for decimal, and D50Ch for duodecimal (base 12). Using this instruction in 64-bit mode generates an invalid-opcode exception.
