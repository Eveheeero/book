FXSAVE and FXRSTOR Image (Non-64-bit Mode) Software can read and write all fields within the FXSAVE and FXRSTOR memory image. These fields include: • FCW—Bytes 01h–00h. x87 control word. • FSW—Bytes 03h–02h. x87 status word. • FTW—Byte 04h. x87 tag word.
