Floating-point and integer data type each have a unique encoding that represents an indefinite value. The processor returns an indefinite value when a masked invalid-operation exception (IE) occurs. For example, if a floating-point division operation is attempted using source operands that are both zero, and IE exceptions are masked, the floating-point indefinite value is returned as the result.
