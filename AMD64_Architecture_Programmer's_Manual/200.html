(V)PSHUFHW Shuffle Operation (V)PSHUFHW and (V)PSHUFLW are useful, for example, in color imaging when computing alpha saturation of RGB values. In this case, (V)PSHUFxW can replicate an alpha value in a register so that parallel comparisons with three RGB values can be performed. AVX2 adds support for 256-bit operands to the AVX forms of these instructions. 4.6.4 Arithmetic Arithmetic operations can be unary or binary. A unary operation has a single operand and produces a single result. A binary operation has two operands that are combined arithmetically to produce a result. A vector arithmetic operation applies the same operation independently to all elements of a vector. Figure 4-31 shows a typical unary vector operation.
