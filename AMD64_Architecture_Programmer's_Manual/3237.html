x87 Floating-Point FCOS 239 Instruction Reference 26569—Rev. 3.16—November 2021 AMD64 Technology Computes the cosine of the radian value in ST(0) and stores the result in ST(0). If the radian value lies outside the valid range of –2 63 to +2 63 radians, the instruction sets the C2 flag in the x87 status word to 1 to indicate the value is out of range and does not change the value in ST(0).
