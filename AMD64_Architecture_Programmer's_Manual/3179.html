64-Bit Media PSRLQ 181 Instruction Reference 26569—Rev. 3.16—November 2021 AMD64 Technology Right-shifts each 64-bit value in the first source operand by the number of bits specified in the second source operand and writes each shifted value in the corresponding quadword of the destination (first source). The first source/destination and second source operands are: • an MMX register and another MMX register or 64-bit memory location, or • an MMX register and an immediate byte value. The high-order bits that are emptied by the shift operation are cleared to 0. If the shift value is greater than 63, the destination is cleared to 0. The PSRLQ instruction is an MMX™ instruction. The presence of this instruction set is indicated by CPUID feature bits. See “CPUID” in Volume 3 for more information about the CPUID instruction.
