B.4 Instructions with 64-Bit Default Operand Size In 64-bit mode, two groups of instructions default to 64-bit operand size without the need for a REX prefix: • Near branches —CALL, Jcc, JrCX, JMP, LOOP, and RET. • All instructions, except far branches, that implicitly reference the RSP—CALL, ENTER, LEAVE, POP, PUSH, and RET (CALL and RET are in both groups of instructions).
