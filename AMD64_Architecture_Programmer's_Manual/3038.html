40 MOVQ2DQ 64-Bit Media Instruction Reference AMD64 Technology 26569—Rev. 3.16—November 2021 Moves a 64-bit value from an MMX register to the low-order 64 bits of an XMM register, with zero- extension to 128 bits. The MOVQ2DQ instruction is an SSE2 instruction. Support for this instruction subset is indicated by CPUID Fn0000_0001_EDX[SSE2] = 1. See “CPUID” in Volume 3 for more information about the CPUID instruction.
