See “Page-Translation-Table Entry Fields” on page 153 for more information on the page-table entries. 7.8.3 Identifying PAT Support Software determines whether a processor supports the PAT mechanism by executing the CPUID instruction with either function 0000_0001h or function 8000_0001h. If PAT is supported, bit 16 in the EDX register is set to 1 by CPUID. See Section 3.3, “Processor Feature Identification,” on page 71 for more information on the CPUID instruction. If PAT is supported by a processor implementation, it is always enabled. The PAT mechanism cannot be disabled by software. Software can effectively avoid using PAT by: • Not setting PAT bits in page-table entries to 1. • Not modifying the reset values of the PA fields in the PAT register. In this case, memory is characterized using the same types that are used by implementations that do not support PAT. 7.8.4 PAT Accesses In implementations that support the PAT mechanism, all memory accesses that are translated through the paging mechanism use the PAT index bits to specify a PA field in the PAT register.
