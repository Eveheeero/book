x87 Floating-Point FBLD 225 Instruction Reference 26569—Rev. 3.16—November 2021 AMD64 Technology Converts a 10-byte packed BCD value in memory into double-extended-precision format, and pushes the result onto the x87 stack. In the process, it preserves the sign of the source value. The packed BCD digits should be in the range 0 to 9. Attempting to load invalid digits (Ah through Fh) produces undefined results.
