Instruction Reference VPHADDBD 761 26568—Rev. 3.25—November 2021 AMD64 Technology Adds four sets of four 8-bit signed integer values of the source and packs the sign-extended sums into the corresponding doubleword of the destination. There are two operands: VPHADDBD dest, src The destination is an XMM register and the source is either an XMM register or a 128-bit memory location. Bits [255:128] of the corresponding YMM register are cleared.
