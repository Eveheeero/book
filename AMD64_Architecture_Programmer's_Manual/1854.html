However, it is not possible to override the operand size to 32 bits because there is no 32-bit operand- size override prefix for 64-bit mode. See “Operand-Size Override Prefix” on page 7 for details. B.5 Single-Byte INC and DEC Instructions in 64-Bit Mode In 64-bit mode, the legacy encodings for the 16 single-byte INC and DEC instructions (one for each of the eight GPRs) are used to encode the REX prefix values, as described in “REX Prefix” on page 14. Therefore, these single-byte opcodes for INC and DEC are not available in 64-bit mode, although they are available in legacy and compatibility modes.
