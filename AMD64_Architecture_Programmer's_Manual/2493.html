532 ROUNDPS, VROUNDPS Instruction Reference AMD64 Technology 26568—Rev. 3.25—November 2021 Rounds four or eight single-precision floating-point values as specified by an immediate byte oper- and. Source values are rounded to integral values and written to the destination as single-precision floating-point values. SNaN source values are converted to QNaN. When DAZ =1, denormals are converted to zero before rounding. The immediate byte operand is defined as follows. Precision exception definitions: Rounding control source definitions: Rounding control definition: There are legacy and extended forms of the instruction: ROUNDPS Rounds four source values. The first source operand is an XMM register. The second source operand is either an XMM register or a 128-bit memory location. There is a third 8-bit immediate operand. The first source register is also the destination. Bits [255:128] of the YMM register that corresponds to the destination are not affected.
