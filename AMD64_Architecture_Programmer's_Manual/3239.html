x87 Floating-Point FDECSTP 241 Instruction Reference 26569—Rev. 3.16—November 2021 AMD64 Technology Decrements the top-of-stack pointer (TOP) field of the x87 status word. If the TOP field contains 0, it is set to 7. In other words, this instruction rotates the stack by one position.
