MCA_INTR_CFG Register 9.3.1.5 CPU Watchdog Timer Register The CPU watchdog timer is used to generate a machine check condition when an instruction does not complete within a time period specified by the CPU Watchdog Timer register. The timer restarts the count each time an instruction completes, when enabled by the CPU Watchdog Timer Enable bit. The time period is determined by the Count Select and Time Base fields. The timer does not count during halt or stop-grant.
