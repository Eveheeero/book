Mnemonic Syntax for Typical Instruction This example shows the FADD mnemonic followed by two operands, both of which are 80-bit stack- register operands. Most instructions take source operands from an x87 stack register and/or memory and write their results to a stack register or memory. Only two of the instructions (FSTSW and FNSTSW) can access a general-purpose registers (GPR), and none access the 128-bit media (XMM) registers. Although the MMX registers map to the x87 registers, the contents of the MMX registers cannot be accessed meaningfully using x87 instructions. Instructions can have one or more prefixes that modify default operand properties.
