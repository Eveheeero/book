E.4.17 Function 8000_001Fh—Encrypted Memory Capabilities CPUID Fn8000_001E_ECX Node Identifiers Bits Field Name Description 31:0 — Reserved 10:8 NodesPerProcessor Specifies the number of nodes in the package/socket in which this logical processor resides.
