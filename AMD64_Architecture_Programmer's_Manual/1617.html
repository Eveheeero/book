350 General-Purpose Instruction Reference AMD64 Technology 24594—Rev. 3.35—June 2023 Performs a bit-wise logical and on the value in a register or memory location (first operand) with an immediate value or the value in a register (second operand) and sets the flags in the rFLAGS register based on the result. This instruction has two operands: TEST dest, src While the AND instruction changes the contents of the destination and the flag bits, the TEST instruction changes only the flag bits.
