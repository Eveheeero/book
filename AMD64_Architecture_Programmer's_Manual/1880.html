Obtaining Processor Information Via the CPUID Instruction 613 24594—Rev. 3.35—June 2023 AMD64 Technology Subfunction 0 provides information about PQM capabilities. Subfunction 1 provides information about PQM L3 Cache Monitoring capabilities. E.3.13 Function 10h—PQOS Enforcement (PQE) If PQE is supported (CPUID Fn0000_0007_EBX_x0[PQE] = 1), this CPUID function can be used to obtain PQE feature information. If PQE is not supported this function is reserved. For more information on using PQE, see “Platform Quality of Service” in APM Volume 2.
