280 FPATAN x87 Floating-Point Instruction Reference AMD64 Technology 26569—Rev. 3.16—November 2021 Computes the arctangent of the ordinate (Y) in ST(1) divided by the abscissa (X) in ST(0), which is the angle in radians between the X axis and the radius vector from the origin to the point (X, Y). It then stores the result in ST(1) and pops the x87 register stack. The resulting value has the same sign as the ordinate value and a magnitude less than or equal to π. There is no restriction on the range of values that FPATAN can accept.
