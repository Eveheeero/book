Secure Virtual Machine 580 24593—Rev. 3.41—June 2023 AMD64 Technology 15.29.9.1 AVIC IPI Delivery Not Completed An IPI could not be delivered to all targeted guest virtual processors because at least one guest virtual processor was not allocated to a physical core at the time. This results in a #VMEXIT with an exit code of AVIC_INCOMPLETE_IPI. Additional data associated with this #VMEXIT event is returned in the EXITINFO1 and EXITINFO2 fields. EXITINFO1.
