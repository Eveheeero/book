372 System Instruction Reference AMD64 Technology 24594—Rev. 3.35—June 2023 Clears the global interrupt flag (GIF). While GIF is zero, all external interrupts are disabled. This is a Secure Virtual Machine instruction. Support for the SVM architecture and the SVM instructions is indicated by CPUID Fn8000_0001_ECX[SVM] = 1. For more information on using the CPUID instruction, see the reference page for the CPUID instruction on page 165. This instruction generates a #UD exception if SVM is not enabled.
