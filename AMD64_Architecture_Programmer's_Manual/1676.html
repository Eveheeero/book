System Instruction Reference 409 24594—Rev. 3.35—June 2023 AMD64 Technology Loads the lower four bits of the 16-bit register or memory operand into bits 3:0 of the machine status word in register CR0. Only the protection enabled (PE), monitor coprocessor (MP), emulation (EM), and task switched (TS) bits of CR0 are modified. Additionally, LMSW can set CR0.PE, but cannot clear it. The LMSW instruction can be used only when the current privilege level is 0. It is only provided for compatibility with early processors. Use the MOV CR0 instruction to load all 32 or 64 bits of CR0.
