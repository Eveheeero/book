Instruction Reference VPERMILPS 743 26568—Rev. 3.25—November 2021 AMD64 Technology Copies single-precision floating-point values from a source to a destination. Source and destination can be selected in two ways. There are different encodings for each selection method. Selection by bit fields in a source register or memory location: Each doubleword of the operand is defined as follows. Each bit field corresponds to a destination doubleword. Bit values select a source doubleword. Only bits [1:0] of each word are used; bits [31:2} are ignored. The 128-bit encoding uses four two-bit fields; the 256-bit version uses eight two-bit fields. Field encoding is as follows.
