530 ROUNDPD, VROUNDPD Instruction Reference AMD64 Technology 26568—Rev. 3.25—November 2021 VROUNDPD The extended form of the instruction has both 128-bit and 256-bit encodings: XMM Encoding Rounds two source values. The first source operand is an XMM register. The second source operand is either an XMM register or a 128-bit memory location. There is a third 8-bit immediate operand. The destination is a third XMM register. Bits [255:128] of the YMM register that corresponds to the destination are cleared. YMM Encoding Rounds four source values. The first source operand is a YMM register and the second source oper- and is either a YMM register or a 256-bit memory location. There is a third 8-bit immediate operand. The destination is a third YMM register. Instruction Support For more on using the CPUID instruction to obtain processor feature support information, see Appen- dix E of Volume 3.
