x87 Floating-Point FBSTP 227 Instruction Reference 26569—Rev. 3.16—November 2021 AMD64 Technology Converts the value in ST(0) to an 18-digit packed BCD integer, stores the result in the specified memory location, and pops the register stack. It rounds a non-integral value to an integer value, depending on the rounding mode specified by the RC field of the x87 control word. The operand specifies the memory address of the first byte of the resulting 10-byte value.
