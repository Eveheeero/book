Advanced Programmable Interrupt Controller (APIC) 620 24593—Rev. 3.41—June 2023 AMD64 Technology 16 Advanced Programmable Interrupt Controller (APIC) The Advanced Programmable Interrupt Controller (APIC) provides interrupt support on AMD64 architecture processors. The local APIC accepts interrupts from the system and delivers them to the local CPU core interrupt handler. Support for APIC is indicated by CPUID Fn0000_0001_EDX[APIC] = 1. For information on using the CPUID instruction to obtain processor implementation information, see Section 3.3, “Processor Feature Identification,” on page 71.
