218 F2XM1 x87 Floating-Point Instruction Reference AMD64 Technology 26569—Rev. 3.16—November 2021 Raises 2 to the power specified by the value in ST(0), subtracts 1, and stores the result in ST(0). The source value must be in the range –1.0 to +1.0. The result is undefined for source values outside this range. This instruction, when used in conjunction with the FYL2X instruction, can be applied to calculate z=x y by taking advantage of the log property x y =2 y*log 2 x .
