Supported 16-Bit Floating-Point Encodings 4.3.3.4 Vector and Scalar Data Types Most SSE instructions accept vector or scalar operands. These data types are composites of the fundamental data types discussed above. The following data types are supported: • Vector (packed) single-precision (32-bit) floating-point numbers • Vector (packed) double-precision (64-bit) floating-point numbers • Vector (packed) signed (two's-complement) integers • Vector (packed) unsigned integers • Scalar single- and double-precision floating-point numbers • Scalar signed (two's-complement) integers • Scalar unsigned integers Hardware does not check or enforce the data types for instructions. Software is responsible for ensuring that each operand for an instruction is of the correct data type.
