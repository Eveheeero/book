Each variant will be discussed below. 4.7.5.1 Operand Source Specification Each instruction operates on three operands to produce a result. Individual instruction forms within a variant allow either the second or third operand to be sourced from memory. In the following descriptions, the first operand will be referred to as operand a, the second will be referred to as operand b and the third operand c. The instruction syntax for specifying this alternate sourcing of the second and third operands differs between the FMA4 and the three operand FMA instructions. The FMA4 instructions use the same instruction mnemonic allowing the memory operand source to appear in either the second or third operand position: VFNM optxx dest_reg src_reg src_regmem src_reg VFNM optxx dest_reg src_reg src_reg src_regmem The three-operand instructions utilize three different instruction mnemonics having the following syntax: VFNM opt xx src_reg src_reg src_regmem VFNM opt xx src_reg src_reg src_regmem VFNM opt xx src_reg src_reg src_regmem Where opt represents the instruction operation and xx represents the operand data type.
