Instruction Reference PCMPESTRI, VPCMPESTRI 309 26568—Rev. 3.25—November 2021 AMD64 Technology The first source operand is an XMM register. The second source operand is either an XMM register or a 128-bit memory location. A result index is written to the ECX register.
