Instruction Reference VPHADDUWD 777 26568—Rev. 3.25—November 2021 AMD64 Technology Adds four adjacent pairs of 16-bit unsigned integer values of the source and packs the sums into the corresponding doubleword of the destination. There are two operands: VPHADDUWD dest, src The destination is an XMM register and the source is either an XMM register or a 128-bit memory location. Bits [255:128] of the corresponding YMM register are cleared.
