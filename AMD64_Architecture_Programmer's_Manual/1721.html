454 System Instruction Reference AMD64 Technology 24594—Rev. 3.35—June 2023 Validates a non-busy (not in-use) shadow stack token pointed to by the PL0_SSP MSR and sets the token’s busy bit. If the validation checks pass, SSP is set to the address in PL0_SSP. SETSSBY is a privileged instruction and must be executed with CPL=0, otherwise a #GP exception is generated. If shadow stacks are not enabled at the supervisor level, a #UD exception is generated.
