172 General-Purpose Instruction Reference AMD64 Technology 24594—Rev. 3.35—June 2023 Subtracts 1 from the specified register or memory location. The CF flag is not affected. The one-byte forms of this instruction (opcodes 48 through 4F) are used as REX prefixes in 64-bit mode. See “REX Prefix” on page 14. The forms of the DEC instruction that write to memory support the LOCK prefix. For details about the LOCK prefix, see “Lock Prefix” on page 11. To perform a decrement operation that updates the CF flag, use a SUB instruction with an immediate operand of 1.
