Multiply-Add Operation The operation shown in Figure 5-4 can be used together with transpose and vector-add operations (see “Addition” on page 262) to accumulate dot product results (also called inner or scalar products), which are used in many media algorithms. 5.3.4 Saturation Several of the 64-bit media integer instructions and most of the 64-bit media floating-point instructions produce vector results in which each element saturates independently of the other elements in the result vector. Such results are clamped (limited) to the maximum or minimum value representable by the destination data type when the true result exceeds that maximum or minimum representable value. Saturation avoids the need for code that tests for potential overflow or underflow. Saturating data is useful for representing physical-world data, such as sound and color.
