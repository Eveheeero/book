122 PFRCPIT2 64-Bit Media Instruction Reference AMD64 Technology 26569—Rev. 3.16—November 2021 Operation mmx1[31:0] = Expand(mmx1[31:0]) * mmx2/mem64[31:0]; mmx1[63:32] = Expand(mmx1[63:32]) * mmx2/mem64[63:32]; where: “Expand” means convert a 24-bit significand to a 32-bit significand according to the following rule: temp[31:0] = {1’b1, 8{mmx1[22]}, mmx1[22:0]}; Examples The general Newton-Raphson recurrence for the reciprocal 1/b is: Z i +1 ← Z i • (2 – b • Z i ) The following code sequence computes a 24-bit approximation to a/b with one Newton-Raphson iteration: X 0 = PFRCP(b) X 1 = PFRCPIT1(b, X 0 ) X 2 = PFRCPIT2(X 1 , X 0 ) q = PFMUL(a, X 2 ) a/b is formed in the last step by multiplying the reciprocal approximation by a.
