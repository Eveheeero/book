220 MOVNTSS Instruction Reference AMD64 Technology 26568—Rev. 3.25—November 2021 Stores one single-precision floating-point value from an XMM register to a 32-bit memory location. This instruction indicates to the processor that the data is non-temporal, and is unlikely to be used again soon. The processor treats the store as a write-combining memory write, which minimizes cache pollution.
