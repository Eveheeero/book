System-Management Mode 330 24593—Rev. 3.41—June 2023 AMD64 Technology 10.2.4 SMM-Revision Identifier The SMM-revision identifier specifies the SMM version and the available SMM extensions implemented by the processor. Software reads the SMM-revision identifier from offset FEFCh in the SMM state-save area of SMRAM. This offset location is compatible with earlier versions of SMM. Software must not write to this location. Doing so can produce undefined results. Figure 10-3 on page 331 shows the format of the SMM-revision identifier.
