262 General-Purpose Instruction Reference AMD64 Technology 24594—Rev. 3.35—June 2023 Does nothing. This instruction increments the rIP to point to next instruction, but does not affect the machine state in any other way. The single-byte variant is an alias for XCHG rAXrAX .
