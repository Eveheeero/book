The combination of the Match bit in each selector element and the value of the M2Z field determines if the Select field is overridden. This is described below. m2z immediate operand The fifth operand is m2z. The assembler uses this 2-bit value to encode the M2Z field in the instruc- tion. M2Z occupies bits [1:0] of an immediate byte. Bits [7:4] of the same byte are used to select one of 16 YMM/XMM registers.
