Instruction Reference VPHSUBBW 785 26568—Rev. 3.25—November 2021 AMD64 Technology Subtracts the most significant signed integer byte from the least significant signed integer byte of each word element in the source and packs the sign-extended 16-bit integer differences into the desti- nation. There are two operands: VPHSUBBW dest, src The destination is an XMM register and the source is either an XMM register or a 128-bit memory location. When the destination is written, bits [255:128] of the corresponding YMM register are cleared.
