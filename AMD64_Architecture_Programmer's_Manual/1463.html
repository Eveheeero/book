196 General-Purpose Instruction Reference AMD64 Technology 24594—Rev. 3.35—June 2023 Checks the overflow flag (OF) in the rFLAGS register and calls the overflow exception (#OF) handler if the OF flag is set to 1. This instruction has no effect if the OF flag is cleared to 0. The INTO instruction detects overflow in signed number addition. See APM Volume 1 for more information on the OF flag. Using this instruction in 64-bit mode generates an invalid-opcode exception. For detailed descriptions of the steps performed by INT instructions, see the following: • Legacy-Mode Interrupts: “Legacy Protected-Mode Interrupt Control Transfers” in APM Volume 2. • Long-Mode Interrupts: “Long-Mode Interrupt Control Transfers” in APM Volume 2.
