254 FINCSTP x87 Floating-Point Instruction Reference AMD64 Technology 26569—Rev. 3.16—November 2021 Increments the top-of-stack pointer (TOP) field of the x87 status word. If the TOP field contains 7, it is cleared to 0. In other words, this instruction rotates the stack by one position.
