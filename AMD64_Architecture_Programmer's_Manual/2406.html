Instruction Reference PSHUFLW, VPSHUFLW 445 26568—Rev. 3.25—November 2021 AMD64 Technology The most-significant quadword of the upper 128 bits of the source is copied to the corresponding quadword of the destination. There are legacy and extended forms of the instruction: PSHUFLW The source operand is either an XMM register or a 128-bit memory location. The destination is an XMM register. Bits [255:128] of the YMM register that corresponds to the destination are not affected. VPSHUFLW The extended form of the instruction has 128-bit and 256-bit encodings. XMM Encoding The source operand is either an XMM register or a 128-bit memory location. The destination is an XMM register. Bits [255:128] of the YMM register that corresponds to the destination are cleared. YMM Encoding The source operand is either a YMM register or a 256-bit memory location. The destination is a YMM register. Instruction Support For more on using the CPUID instruction to obtain processor feature support information, see Appen- dix E of Volume 3.
