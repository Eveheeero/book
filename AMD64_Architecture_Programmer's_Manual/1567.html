300 General-Purpose Instruction Reference AMD64 Technology 24594—Rev. 3.35—June 2023 Loads the destination register with a hardware-generated random “seed” value. The size of the returned value in bits is determined by the size of the destination register. Hardware modifies the CF flag to indicate whether the value returned in the destination register is valid. If CF = 1, the value is valid. If CF = 0, the value is invalid and will be returned as zero. Software must test the state of the CF flag prior to using the value returned in the destination register to determine if the value is valid. If the returned value is invalid, software must execute the instruction again. Software should implement a retry limit to ensure forward progress of code.
