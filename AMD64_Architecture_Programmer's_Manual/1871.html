604 Obtaining Processor Information Via the CPUID Instruction AMD64 Technology 24594—Rev. 3.35—June 2023 The value returned in EAX is undefined and is reserved. The value returned in EBX is undefined and is reserved. The value returned in ECX indicates support of the processor effective frequency interface. For more information on this feature, see “Determining Processor Effective Frequency” in APM Volume 2. The value returned in EDX is undefined and is reserved.
