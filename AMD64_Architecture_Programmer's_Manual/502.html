47 System Resources AMD64 Technology 24593—Rev. 3.41—June 2023 Figure 3-7. CR4 Register The function of the CR4 control bits are (all bits are read/write): Virtual-8086 Mode Extensions (VME). Bit 0. Setting VME to 1 enables hardware-supported performance enhancements for software running in virtual-8086 mode. Clearing VME to 0 disables this support.
