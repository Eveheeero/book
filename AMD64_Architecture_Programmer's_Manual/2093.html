132 DPPD, VDPPD Instruction Reference AMD64 Technology 26568—Rev. 3.25—November 2021 There are legacy and extended forms of the instruction: DPPD The first source operand is an XMM register. The second source operand is either an XMM register or a 128-bit memory location. The first source register is also the destination. Bits [255:128] of the YMM register that corresponds to the destination are not affected. VDPPD The extended form of the instruction has a single 128-bit encoding. The first source operand is an XMM register. The second source operand is either an XMM register or a 128-bit memory location. The destination is a third XMM register. Bits [255:128] of the YMM reg- ister that corresponds to the destination are cleared.
