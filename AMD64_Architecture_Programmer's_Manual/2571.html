610 VEXTRACTF128 Instruction Reference AMD64 Technology 26568—Rev. 3.25—November 2021 Extracts 128 bits of packed data from a YMM register as specified by an immediate byte operand, and writes it to either an XMM register or a 128-bit memory location. Only bit [0] of the immediate operand is used. Operation is as follows. • When imm8[0] = 0, copy bits [127:0] of the source to the destination. • When imm8[0] = 1, copy bits [255:128] of the source to the destination. This extended-form instruction has a single 256-bit encoding. The source operand is a YMM register and the destination is either an XMM register or a 128-bit memory location. There is a third immediate byte operand. Instruction Support For more on using the CPUID instruction to obtain processor feature support information, see Appen- dix E of Volume 3.
