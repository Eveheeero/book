Instruction Reference VBROADCASTF128 595 26568—Rev. 3.25—November 2021 AMD64 Technology Loads double-precision floating-point data from a 128-bit memory location and writes it to the two 128-bit elements of a YMM register This extended-form instruction has a single 256-bit encoding. The source operand is a 128-bit memory location. The destination is a YMM register.
