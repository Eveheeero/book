Instruction Reference VBROADCASTSD 599 26568—Rev. 3.25—November 2021 AMD64 Technology Loads a double-precision floating-point value from a register or memory and writes it to the four 64- bit elements of a YMM register This extended-form instruction has a single 256-bit encoding. The source operand is the lower half of an XMM register or a 64-bit memory location. The destina- tion is a YMM register.
