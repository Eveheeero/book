220 FABS x87 Floating-Point Instruction Reference AMD64 Technology 26569—Rev. 3.16—November 2021 Converts the value in ST(0) to its absolute value by clearing the sign bit. The resulting value depends upon the type of number used as the source value: This operation applies even if the value in ST(0) is negative zero or negative infinity.
