376 System Instruction Reference AMD64 Technology 24594—Rev. 3.35—June 2023 Validates the busy (in use) shadow stack token pointed to by the memory operand and clears the token’s busy bit. If the token validation checks pass, CF is cleared to 0 and SSP is cleared to 0. If the token validation checks fail, CF is set to 1 and the token and SSP are not modified. CLRSSBY is a privileged instruction and must be executed with CPL=0, otherwise a #GP exception is generated. If shadow stacks are not enabled at the supervisor level, a #UD exception is generated.
