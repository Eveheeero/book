Instruction Reference MOVSD, VMOVSD 225 26568—Rev. 3.25—November 2021 AMD64 Technology Instruction Encoding Related Instructions (V)MOVAPD, (V)MOVHPD, (V)MOVLPD, (V)MOVMSKPD, (V)MOVUPD Mnemonic Opcode Description MOVSD xmm1, xmm2/mem64 F2 0F 10 /r Moves a 64-bit value from xmm2 or mem64 to xmm1. Zero extends to 128 bits when source operand is memory. MOVSD xmm1/mem64, xmm2 F2 0F 11 /r Moves a 64-bit value from xmm2 to xmm1 or mem64.
