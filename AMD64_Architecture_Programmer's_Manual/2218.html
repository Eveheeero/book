Instruction Reference PABSW, VPABSW 257 26568—Rev. 3.25—November 2021 AMD64 Technology Computes the absolute value of eight or sixteen packed 16-bit signed integers in the source operand. Each word of the destination receives an unsigned 16-bit integer that is the absolute value of the signed 16-bit integer in the corresponding word of the source operand. There are legacy and extended forms of the instruction: PABSW The source operand is an XMM register or a 128-bit memory location. The destination is an XMM register. Bits [255:128] of the YMM register that corresponds to the destination are not affected. VPABSW The extended form of the instruction has 128-bit and 256-bit encodings. XMM Encoding The source operand is an XMM register or a 128-bit memory location. The destination is an XMM register. Bits [255:128] of the YMM register that corresponds to the destination are cleared. YMM Encoding The source operand is a YMM register or a 256-bit memory location. The destination is a YMM reg- ister. All 16 words of the destination are written.
