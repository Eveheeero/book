738 VPERMIL2PS Instruction Reference AMD64 Technology 26568—Rev. 3.25—November 2021 src2 and src3 Operand Addressing In 64-bit mode, VEX.W and bits [7:4] of the immediate byte specify src2 and src3: • When VEX.W = 0, src2 is either a register or a memory location specified by ModRM.r/m and src3 is a register specified by bits [7:4] of the immediate byte. • When VEX.W = 1, src2 is a register specified by bits [7:4] of the immediate byte and src3 is either a register or a memory location specified by ModRM.r/m. In non-64-bit mode, bit 7 is ignored. Instruction Support For more on using the CPUID instruction to obtain processor feature support information, see Appen- dix E of Volume 3.
