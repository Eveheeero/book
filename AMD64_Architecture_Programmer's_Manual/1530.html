General-Purpose 263 Instruction Reference 24594—Rev. 3.35—June 2023 AMD64 Technology Performs the one’s complement negation of the value in the specified register or memory location by inverting each bit of the value. The memory-operand forms of the NOT instruction support the LOCK prefix. For details about the LOCK prefix, see “Lock Prefix” on page 11.
