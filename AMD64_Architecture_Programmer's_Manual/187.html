Streaming SIMD Extensions Media and Scientific Programming 151 24592—Rev. 3.23—October 2020 AMD64 Technology The MOVDQ2Q instruction copies the low-order 64-bit value in an XMM register to an MMX register. The MOVQ2DQ instruction copies a 64-bit value from an MMX register to the low-order 64 bits of an XMM register, with zero-extension to 128 bits.
