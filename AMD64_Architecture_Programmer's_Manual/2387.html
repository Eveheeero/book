426 PMULLD, VPMULLD Instruction Reference AMD64 Technology 26568—Rev. 3.25—November 2021 Multiplies four packed 32-bit signed integers in the first source operand by the corresponding values in the second source operand and writes bits [31:0] of each 64-bit product to the corresponding 32-bit element of the destination. There are legacy and extended forms of the instruction: PMULLD The first source operand is an XMM register. The second source operand is either an XMM register or a 128-bit memory location. The first source register is also the destination. Bits [255:128] of the YMM register that corresponds to the destination are not affected. VPMULLD The extended form of the instruction has 128-bit and 256-bit encodings. XMM Encoding The first source operand is an XMM register. The second source operand is either an XMM register or a 128-bit memory location. The destination is a third XMM register. Bits [255:128] of the YMM reg- ister that corresponds to the destination are cleared. YMM Encoding The first source operand is a YMM register. The second source operand is either a YMM register or a 256-bit memory location. The destination is a third YMM register.
