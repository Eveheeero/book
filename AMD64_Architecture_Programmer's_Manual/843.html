A general-protection exception (#GP) occurs if an attempt is made to switch to a task with a set busy bit. • An IRET to another task (EFLAGS.NT must be 1) clears the busy bit in the old task to 0. The busy bit in the new task is not altered, because it was already set to 1. Table 12-1 on page 387 summarizes the effect various task-switch initiators have on the TSS-busy bit. [AMD Public Use]
