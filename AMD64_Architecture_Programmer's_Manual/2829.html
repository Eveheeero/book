868 AMD64 Technology 26568—Rev. 3.25—November 2021 Restores a selected set of enabled processor state data from a save area at a specified address in memory, optionally including privileged state. XRSTORS is very similar to the XRSTOR instruction in compacted form with the following differences: 1. XRSTORS must be executed at CPL=0 2. XRSTORS must read XCOMP_BV[63]=1, otherwise it will cause a #GP(0) exception 3. XRSTORS is able to restore state enabled from the IA32_XSS MSR. All other behavior is the same as XRSTOR with the compact form.
