34 MOVDQ2Q 64-Bit Media Instruction Reference AMD64 Technology 26569—Rev. 3.16—November 2021 Moves the low-order 64-bit value in an XMM register to a 64-bit MMX register. The MOVDQ2Q instruction is an SSE2 instruction. Support for this instruction subset is indicated by CPUID Fn0000_0001_EDX[SSE2] = 1. See “CPUID” in Volume 3 for more information about the CPUID instruction.
