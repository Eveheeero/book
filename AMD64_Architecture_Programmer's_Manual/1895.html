E.4.9 Function 8000_000Ah—SVM Features This function provides information about the SVM features that the processor supports. If SVM is not supported (CPUID Fn8000_0001_ECX[SVM] = 0), this function is reserved. The value returned in EAX provides the SVM revision number. The value returned in EBX provides the number of address space identifiers (ASIDs) that the processor supports. The value returned in ECX for this function is undefined and is reserved. The value returned in EDX provides Secure Virtual Machine architecture feature information.
