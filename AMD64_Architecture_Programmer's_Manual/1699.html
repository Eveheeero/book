432 System Instruction Reference AMD64 Technology 24594—Rev. 3.35—June 2023 Reads the current Shadow Stack Pointer (SSP) to the specified GPR. The operand size is 64 bits in 64- bit mode when REX.W=1 and is 32 bits in all other cases. RDSSP is treated as a NOP if CR4.CET = 0, or if shadow stacks are not enabled at the current privilege level.
