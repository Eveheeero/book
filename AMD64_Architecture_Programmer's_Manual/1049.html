The SEV_STATUS MSR is available on processors that support SEV. 15.34.11 Virtual Transparent Encryption (VTE) The Virtual Transparent Encryption feature can be enabled to force all memory accesses within an SEV guest to be encrypted with the guest’s key. Support for this feature is indicated in CPUID Fn8000_001F[EAX] bit 16. To enable this feature, the hypervisor must set VMCB offset 90h bit 5. Bit 5 is only observed when SEV (bit 1) is also set to 1 and SEV-ES (bit 2) is cleared to 0. In all other configurations of these bits (namely SEV disabled or SEV-ES enabled), bit 5 is ignored by hardware. Table 15-34.
