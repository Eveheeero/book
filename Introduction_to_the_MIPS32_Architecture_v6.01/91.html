Removed in Release 6. The UNFR register allows user-mode to set Status FR by executing a CTC1 to UNFR with GPR[0] as input. CTC1 to UNFR with any other input register is required to produce a Reserved Instruction exception. User-mode software can determine presence of this feature from FIR UFRP .
