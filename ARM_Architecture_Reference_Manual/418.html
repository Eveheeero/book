ARM DDI 0100I Operation if ConditionPassed(cond) then Rd[15:0] = UnsignedSat(Rm[15:0], sat_imm) // Rm[15:0] treated as signed Rd[31:16] = UnsignedSat(Rm[31:16], sat_imm) // Rm[31:16] treated as signed if UnsignedDoesSat(Rm[15:0], sat_imm) OR UnsignedDoesSat(Rm[31:16], sat_imm) then Q Flag = 1 Usage You can use USAT16 in various DSP algorithms, such as calculating a pixel color component, that require saturation of signed data to an unsigned destination. Notes Use of R15 Specifying R15 for register <Rd> or <Rm> has UNPREDICTABLE results.
