ARM DDI 0100I A4.1.124 UHSUB16 UHSUB16 (Unsigned Halving Subtract) performs two 16-bit unsigned integer subtractions, and halves the results. It has no effect on the GE flags. Syntax UHSUB16{<cond>} <Rd>, <Rn>, <Rm> where: <cond> Is the condition under which the instruction is executed. The conditions are defined in The condition field on page A3-3. If <cond> is omitted, the AL (always) condition is used. <Rd> Specifies the destination register. <Rn> Specifies the register that contains the first operand. <Rm> Specifies the register that contains the second operand. Architecture version ARMv6 and above. Exceptions None. Operation if ConditionPassed(cond) then diff = Rn[15:0] - Rm[15:0] /* Unsigned subtraction */ Rd[15:0] = diff[16:1] diff = Rn[31:16] - Rm[31:16] /* Unsigned subtraction */ Rd[31:16] = diff[16:1] Usage Use UHSUB16 for similar purposes to USUB16 (see USUB16 on page A4-269). UHSUB16 gives half the difference instead of the full difference. Notes Use of R15 Specifying R15 for register <Rd>, <Rm>, or <Rn> has UNPREDICTABLE results. 31 28 27 26 25 24 23 22 21 20 19 16 15 12 11 8 7 6 5 4 3 0 cond 0 1 1 0 0 1 1 1 Rn Rd SBO 0 1 1 1 Rm
