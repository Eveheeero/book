Each memory region can be tagged as not containing executable code. If the Execute-Never (XN) bit is set to 1, any attempt to execute an instruction in that region results in a permission fault. If the XN bit is cleared to 0, code can execute from that memory region. Note The XN bit acts as an additional permission check. The address must also have a valid read access.
