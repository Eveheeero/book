A7-89 A7.1.52 REV16 REV16 (Byte-Reverse Packed Halfword) reverses the byte order in each 16-bit halfword of a 32-bit register. It does not affect the flags Syntax REV16 Rd, Rn where: <Rd> Specifies the destination register. <Rn> Specifies the register that contains the operand. Architecture version ARMv6 and above. Exceptions None. Operation Rd[15: 8] = Rn[ 7: 0] Rd[ 7: 0] = Rn[15: 8] Rd[31:24] = Rn[23:16] Rd[23:16] = Rn[31:24] Usage Use REV16 to convert 16-bit big-endian data into little-endian data, or 16-bit little-endian data into big-endian data. Equivalent ARM syntax and encoding REV16 Rd, Rm 15 12 11 8 7 6 5 3 2 0 1 0 1 1 1 0 1 0 0 1 Rn Rd 31 28 27 23 22 21 20 19 16 15 12 11 8 7 6 4 3 0 1 1 1 0 0 1 1 0 1 0 1 1 SBO Rd SBO 1 0 1 1 Rm
