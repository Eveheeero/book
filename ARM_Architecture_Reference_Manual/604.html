It does not affect the flags. Syntax REV Rd, Rn where: <Rd> Specifies the destination register. <Rn> Specifies the register that contains the operand. Architecture version ARMv6 and above. Exceptions None. Operation Rd[31:24] = Rn[ 7: 0] Rd[23:16] = Rn[15: 8] Rd[15: 8] = Rn[23:16] Rd[ 7: 0] = Rn[31:24] Usage Use REV to convert 32-bit big-endian data into little-endian data, or 32-bit little-endian data into big-endian data.
