B4-1 Chapter B4 Virtual Memory System Architecture This chapter describes the Virtual Memory System Architecture (VMSA) based on a Memory Management Unit (MMU). It contains the following sections: • About the VMSA on page B4-2 • Memory access sequence on page B4-4 • Memory access control on page B4-8 • Memory region attributes on page B4-11 • Aborts on page B4-14 • Fault Address and Fault Status registers on page B4-19 • Hardware page table translation on page B4-23 • Fine page tables and support of tiny pages on page B4-35 • CP15 registers on page B4-39.
