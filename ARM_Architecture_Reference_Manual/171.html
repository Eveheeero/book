A4-21 A4.1.11 BXJ BXJ (Branch and change to Jazelle ® state) enters Jazelle state if Jazelle is available and enabled. Otherwise BXJ behaves exactly as BX (see BX on page A4-20). Syntax BXJ{<cond>} <Rm> where: <cond> Is the condition under which the instruction is executed. The conditions are defined in The condition field on page A3-3. If <cond> is omitted, the AL (always) condition is used. <Rm> Holds the value of the branch target address for use if Jazelle state is not available. Bit[0] of Rm is 0 to select a target ARM instruction, or 1 to select a target Thumb instruction. Architecture version Version 6 and above, plus ARMv5TEJ.
