ARM DDI 0100I D3.3.2 Register 1, Debug Status and Control Register (DSCR) Table D3-4 shows the layout of the Debug Status and Control Register. Core halted, bit[0] After programming a Debug Event, the external debugger should poll this bit until it is set to 1 so that it knows that the processor has entered Debug state. See Halting debug-mode on page D2-8 for a definition of Debug state. 0 The processor is in normal state. 1 The processor is in Debug state.
