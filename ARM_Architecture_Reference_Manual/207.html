A4-57 Usage Use LDRSB with a suitable addressing mode to load 8-bit signed memory data into a general-purpose register where it can be manipulated. You can perform PC-relative addressing by using the PC as the base register. This facilitates position-independent code. Notes Operand restrictions If <addressing_mode> specifies base register write-back, and the same register is specified for <Rd> and <Rn>, the results are UNPREDICTABLE. Data Abort For details of the effects of the instruction if a Data Abort occurs, see Effects of data-aborted instructions on page A2-21.
