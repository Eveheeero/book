B5-1 Chapter B5 Protected Memory System Architecture This chapter describes the Protected Memory System Architecture (PMSA) based on a Memory Protection Unit (MPU). It contains the following sections: • About the PMSA on page B5-2 • Memory access sequence on page B5-4 • Memory access control on page B5-8 • Memory access attributes on page B5-10 • Memory aborts (PMSAv6) on page B5-13 • Fault Status and Fault Address register support on page B5-16 • CP15 registers on page B5-18.
