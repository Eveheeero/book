A4-273 Usage USUBADDX produces the same result as SSUBADDX (see SSUBADDX on page A4-184), but produces GE bit values based on unsigned arithmetic instead of signed arithmetic. Notes Use of R15 Specifying R15 for register <Rd>, <Rm>, or <Rn> has UNPREDICTABLE results.
