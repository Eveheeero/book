This is often used to form a bit mask. MVN updates the condition code flags, based on the result. Syntax MVN <Rd>, <Rm> where: <Rd> Is the destination register for the operation. <Rm> Is the register containing the value whose ones complement is written to <Rd>. Architecture version All T variants. Exceptions None.
