It is optional which write allocation policies an implementation supports. The allocate on write and no allocate on write cache policies indicate which allocation policy is preferred for a memory region, but it should not be relied on that the memory system implements that policy. Not all inner and outer cache policies are mandatory. Table B4-5 describes the implementation options.
