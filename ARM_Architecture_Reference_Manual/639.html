A7-123 A7.1.73 UXTB UXTB (Unsigned Extend Byte) extracts the least significant 8 bits of the operand, and zero extends the value to 32 bits. Syntax UXTB <Rd>, <Rm> where: <Rd> Specifies the destination register. <Rm> Specifies the operand register. Architecture version ARMv6 and above. Exceptions None. Operation Rd = Rm AND 0x000000ff Usage Use UXTB to zero extend a halfword to a word, for example in instruction sequences acting on unsigned short values in C/C++.
