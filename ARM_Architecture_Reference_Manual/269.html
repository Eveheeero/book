ARM Instructions ARM DDI 0100I Copyright Â© 1996-1998, 2000, 2004, 2005 ARM Limited. All rights reserved. A4-119 A4.1.62 SADD16 SADD16 (Signed Add) performs two 16-bit signed integer additions. It sets the GE bits in the CPSR according to the results of the additions. Syntax SADD16{<cond>} <Rd>, <Rn>, <Rm> where: <cond> Is the condition under which the instruction is executed. The conditions are defined in The condition field on page A3-3. If <cond> is omitted, the AL (always) condition is used. <Rd> Specifies the destination register. <Rn> Specifies the register that contains the first operand. <Rm> Specifies the register that contains the second operand. Architecture version ARMv6 and above. Exceptions None.
