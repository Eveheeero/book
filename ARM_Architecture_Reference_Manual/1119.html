D3-25 D3.5 Access to CP14 debug registers from the external debug interface All the CP14 debug registers that have been defined in this section must be accessible at the External Debug Interface. They must be accessible regardless of the processor state (ARM/Thumb/Jazelle/Debug). In tables throughout this chapter where an External view column is defined, those must be the read/write attributes at the External Debug Interface. Where there is only a Read/write column, those are the attributes for both the core and the external debugger.
