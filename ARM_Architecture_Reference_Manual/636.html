ARM DDI 0100I A7.1.70 SXTB SXTB (Signed Extend Byte) extracts the least significant 8 bits of the operand, and sign extends the value to 32 bits. It does not affect the flags. Syntax SXTB <Rd>, <Rm> where: <Rd> Specifies the destination register. <Rm> Specifies the operand register. Architecture version ARMv6 and above. Exceptions None. Operation Rd = SignExtend(Rm[7:0]) Usage Use SXTB to sign extend a byte to a word, for example in instruction sequences acting on signed char values in C/C++.
