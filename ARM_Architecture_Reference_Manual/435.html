A4-285 Operation if ConditionPassed(cond) then Rd[31:0] = (Rm Rotate_Right(8 * rotate)) AND 0x0000ffff Usage Use UXTH to zero extend a halfword to a word, for example in instruction sequences acting on unsigned short values in C/C++. Notes Use of R15 Specifying R15 for register <Rd> or <Rm> has UNPREDICTABLE results
