The encoding of the size field is shown in Table B7-2. SC (Bit[1]) This TCM is enabled as SmartCache (if the TCM supports SmartCache): 0 = Local RAM (reset state) 1 = SmartCache If the RAM does not support SmartCache, the bit is read as zero, and is ignored on writes. This can be used to determine if the TCM supports SmartCache. En (Bit[0]) 0 = Disabled (Reset state) 1 = Enabled If a TCM is not implemented, then the TCM Region register for that TCM is SBZ/UNP. The implementation-specific representation of the sizes is shown in Table B7-2. The base address of each TCM must be different, such that no location in memory is contained in more than one TCM. If a location in memory is contained in more than one TCM, it is UNPREDICTABLE which memory the instruction or data is returned from. Implementations must ensure that this situation cannot result in physical damage to the TCM.
