ARM DDI 0100I A6.3 Branch instructions Thumb supports six types of branch instruction: • a conditional branch to allow forward and backward branches of up to 256 bytes (-256 to + 254) • an unconditional branch that allows a forward or backward branch of up to 2KB (-2048 to +2046) • a Branch with Link (subroutine call) is supported with a pair of instructions that allow forward and backward branches of up to 4MB (-222 <= offset <= +222 - 2) • a Branch with Link and Exchange uses a pair of instructions, similar to Branch with Link, but additionally switches to ARM code execution. • a Branch and Exchange instruction branches to an address in a register and optionally switches to ARM code execution • a second form of Branch with Link and Exchange instruction performs a subroutine call to an address in a register and optionally switches to ARM code execution The encoding for these instructions is given below.
