B7-9 B7.5 L1 DMA control using CP15 Register 11 The L1 DMA is controlled using CP15 Register 11. There are several registers associated with CP15 Register 11, that are used for the control and monitoring of the DMA. They are defined in Table B7-3. The instructions are: MCR p15, 0, Rd, c11, CRm, Opcode2 MRC p15, 0, Rd, c11, CRm, Opcode2 where CRm is the associated register, and Rd is the ARM ® source or destination register. These, and Opcode2, are as shown in Table B7-3. Further details are given in the following subsections. The Enable, Control, Internal Start Address, External Start Address, Internal End Address, Channel Status, and Context ID registers are multiple registers, with one register of each existing for each channel that is implemented. Which register is accessed is determined by the Channel Number register, as described in TCM Region Registers using CP15 Register 9 on page B7-4. B7.5.1 User Access to Cp15 Register 11 operations A number of CP15 Register 11 operations can be executed by code while in User mode. Attempting to execute a privileged operation in User mode using CP15 register 11 results in an Undefined Instruction exception.
