ARM DDI 0100I A2.3 Registers The ARM processor has a total of 37 registers: • Thirty-one general-purpose registers, including a program counter. These registers are 32 bits wide and are described in General-purpose registers on page A2-6. • Six status registers. These registers are also 32 bits wide, but only some of the 32 bits are allocated or need to be implemented. The subset depends on the architecture variant supported. These are described in Program status registers on page A2-11. Registers are arranged in partially overlapping banks, with the current processor mode controlling which bank is available, as shown in Figure A2-1 on page A2-5. At any time, 15 general-purpose registers (R0 to R14), one or two status registers, and the program counter are visible. Each column of Figure A2-1 on page A2-5 shows which general-purpose and status registers are visible in the indicated processor mode.
