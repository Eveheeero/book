ARM DDI 0100I B5.6 Fault Status and Fault Address register support One or two FSRs are used depending on whether they are reporting faults on a unified or separate (Harvard) instruction and data address space memory model. Three FARs are used to report address information in different contexts. The encodings for the FSR are given in Table B5-8. All other FSR encodings are RESERVED. Table B5-9 on page B5-17 provides a summary of which abort vector is taken, and which of the FSRs and FARs are updated on each abort type.
