A4-261 A4.1.136 USAD8 USAD8 (Unsigned Sum of Absolute Differences) performs four unsigned 8-bit subtractions, and adds the absolute values of the differences together. Syntax USAD8{<cond>} <Rd>, <Rm>, <Rs> where: <cond> Is the condition under which the instruction is executed. The conditions are defined in The condition field on page A3-3. If <cond> is omitted, the AL (always) condition is used. <Rd> Specifies the destination register. <Rm> Specifies the register that contains the first operand. <Rs> Specifies the register that contains the second operand.
