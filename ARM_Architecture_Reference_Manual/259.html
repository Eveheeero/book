ARM Instructions ARM DDI 0100I Copyright Â© 1996-1998, 2000, 2004, 2005 ARM Limited. All rights reserved. A4-109 A4.1.56 REV REV (Byte-Reverse Word) reverses the byte order in a 32-bit register. Syntax REV{<cond>} Rd, Rm where: <cond> Is the condition under which the instruction is executed. The conditions are defined in The condition field on page A3-3. If <cond> is omitted, the AL (always) condition is used. <Rd> Specifies the destination register. <Rm> Specifies the register that contains the operand. Architecture version Version 6 and above. Exceptions None. Operation if ConditionPassed(cond) then Rd[31:24] = Rm[ 7: 0] Rd[23:16] = Rm[15: 8] Rd[15: 8] = Rm[23:16] Rd[ 7: 0] = Rm[31:24] Usage Use REV to convert 32-bit big-endian data into little-endian data, or 32-bit little-endian data into big-endian data. Notes Use of R15 Specifying R15 for register <Rd> or <Rm> has UNPREDICTABLE results. 31 28 27 23 22 21 20 19 16 15 12 11 8 7 6 4 3 0 cond 0 1 1 0 1 0 1 1 SBO Rd SBO 0 0 1 1 Rm
