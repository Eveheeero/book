B2-7 B2.3 Memory attributes prior to ARMv6 Prior to ARMv6, all memory has been tagged with a combination of two control bits in the ARM virtual and protected memory management models, VMSA and PMSA respectively. The bits are: • a bufferable (B) bit (allow write buffering between the core and memory) • a cacheable (C) bit. These are traditionally interpreted to define the memory behavior of a given location as shown in Table B2-1.
