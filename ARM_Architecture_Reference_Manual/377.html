A4-227 Operation if ConditionPassed(cond) then operand2 = Rm Rotate_Right(8 * rotate) Rd[31:0] = SignExtend(operand2[15:0]) Usage Use SXTH to sign-extend a halfword to a word, for example in instruction sequences acting on signed short values in C/C++. Notes Use of R15 Specifying R15 for register <Rd> or <Rm> has UNPREDICTABLE results
