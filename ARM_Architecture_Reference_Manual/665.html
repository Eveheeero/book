B2-9 Table B2-2 Memory attribute summary Memory type attribute Shared attribute Other attributes Description Strongly Ordered - All memory accesses to Strongly Ordered memory occur in program order. All Strongly Ordered accesses are assumed to be Shared. Device Shared Designed to handle memory mapped peripherals that are shared by several processors. Non-Shared Designed to handle memory mapped peripherals that are used only by a single processor. Normal Shared Non-cacheable/ Write-Through cacheable/ Write-Back cacheable Designed to handle normal memory which is shared between several processors.
