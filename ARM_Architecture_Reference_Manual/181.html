It is a synonym for MOV, with no flag setting and no shift. See MOV on page A4-68. Syntax CPY{<cond>} <Rd>, <Rm> where: <cond> Is the condition under which the instruction is executed. The conditions are defined in The condition field on page A3-3. If <cond> is omitted, the AL (always) condition is used. <Rd> Specifies the destination register. <Rm> Specifies the source register. Architecture version Version 6 and above. Exceptions None.
