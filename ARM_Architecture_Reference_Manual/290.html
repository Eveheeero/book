ARM DDI 0100I Usage Use SHSUBADDX for similar purposes to SSUBADDX, but when you want the results halved. See SSUBADDX on page A4-184 for further details. SHSUBADDX does not set any flags, as overflow is not possible. Notes Use of R15 Specifying R15 for register <Rd>, <Rm>, or <Rn> has UNPREDICTABLE results.
