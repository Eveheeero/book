B7-17 MRC p15, 0, Rd, c11, c8, 0 Status (bits[1:0]) 00 = Idle 01 = Queued 10 = Running 11 = Complete/Error. IS (bits[6:2]) Internal Address Error Status, with the following encoding: 0b00xxx = No Error (reset value) 0b01000 = TCM out of range 0b11100 =External Abort on Translation of 1st Level Page Table 0b11110 = External Abort on Translation of 2nd Level Page Table 0b10101 = Translation fault (Section) 0b10111 = Translation fault (Page) 0b11001 = Domain fault (Section) 0b11011 = Domain fault (Page) 0b11101 = Permission fault (Section) 0b11111 = Permission fault (Page). All other encodings are Reserved. ES (bits[11:7]) External Address Error Status, with the following encoding: 0b00xxx = No Error (reset value) 0b01001 = Unshared Data Error 0b11010 = External Abort (can be imprecise) 0b11100 = External Abort on Translation of 1st Level Page Table 0b11110 = External Abort on Translation of 2nd Level Page Table 0b10101 = Translation fault (Section) 0b10111 = Translation fault (Page) 0b11001 = Domain fault (Section) 0b11011 = Domain fault (Page) 0b11101 = Permission fault (Section). 0b11111 = Permission fault (Page). All other encodings are Reserved. If an error occurs, the faulting address is contained in the appropriate start address register, unless the error is an External Error (ES == 0b11010).
