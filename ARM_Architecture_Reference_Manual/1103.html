D3-9 D3.3.1 Register 0, Debug ID Register (DIDR) Table D3-3 shows the layout of the Debug ID Register. The values of the fields of this register are IMPLEMENTATION DEFINED. However, the chosen values must agree with the ones in the CP15 Main ID Register: • DIDR[3:0] must equal the Main ID Register bits[3:0] • DIDR[7:4] must equal the Main ID Register bits[23:20]. See Register 0: ID codes on page B3-7 for a description of the Main ID Register.
