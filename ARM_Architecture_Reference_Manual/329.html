A4-179 Operation if ConditionPassed(cond) then Rd[15:0] = SignedSat(Rm[15:0], sat_imm + 1) Rd[31:16] = SignedSat(Rm[31:16], sat_imm + 1) if SignedDoesSat(Rm[15:0], sat_imm + 1) OR SignedDoesSat(Rm[31:16], sat_imm + 1) then Q Flag = 1 Usage You can use SSAT16 in various DSP algorithms that require saturation of signed data. Notes Use of R15 Specifying R15 for register <Rd> or <Rm> has UNPREDICTABLE results.
