ARM DDI 0100I Notes Assembler syntax If a low register is specified for both <Rd> and <Rm>, the assembler syntax MOV <Rd>, <Rm> is assembled to the MOV (2) instruction described on page A7-73. Both registers low If H1==0 and H2==0 in the encoding, the instruction specifies a non-flag-setting copy move from one low register to another low register. This instruction cannot be written using the MOV syntax, because MOV <Rd>, <Rm> generates a flag-setting copy. However, you can write it using the CPY mnemonic, see CPY on page A7-41. Note Prior to ARMv6, specifying a low register for <Rd> and <Rm> (H1 == 0 and H2 == 0), the result is UNPREDICTABLE.
