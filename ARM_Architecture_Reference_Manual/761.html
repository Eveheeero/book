B5-9 B5.3.2 Data access permissions (PMSAv6) In PMSAv6, the access permission bits are extended to a 3-bit field, and control access to the corresponding memory region. If an access is made to an area of memory without the required permissions, a Permission Fault is raised. The access permissions are determined by the AP bits in the data access permission registers. B5.3.3 Instruction access permissions (PMSAv6) Separate access permissions are supported for instruction accesses. This allows areas of memory to be marked as non-executable, that is, contain data only, without affecting data accesses. For instructions to be executed from a memory region, the region must have data read access (indicated by AP[2:0]) and the XN bit shall be 0.
