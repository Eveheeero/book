The value contained in the channel number register is used to determine which of the multiple registers is accessed when one of these registers is specified. This register can be accessed by user processes if the U Bit of any channel is set to 1. Attempting to access them by a user process if no channel has the U bit set to 1 results in an Undefined Instruction exception.
