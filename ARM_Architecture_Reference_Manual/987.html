C4-71 Notes Conversions In the programmer's model, FMSRR does not perform any conversion of the values transferred. Arithmetic instructions on either of the ARM registers treat the contents as an integer. Most VFP instructions treat the Sm and Sn values as single-precision floating-point numbers. Invalid register lists If S31 is specified as Sm, the results are UNPREDICTABLE. If the register pair is not consecutive, an error is reported by the assembler.
