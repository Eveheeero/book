A7-121 A7.1.71 SXTH SXTH16 (Signed Extend Halfword) extracts the least significant 16 bits of the operand, and sign extends the value to 32 bits. SXTH does not affect the flags. Syntax SXTH <Rd>, <Rm> where: <Rd> Specifies the destination register. <Rm> Specifies the operand register. Architecture version ARMv6 and above. Exceptions None. Operation Rd = SignExtend(Rm[15:0]) Usage Use SXTH to sign extend a halfword to a word, for example in instruction sequences acting on signed short values in C/C++.
