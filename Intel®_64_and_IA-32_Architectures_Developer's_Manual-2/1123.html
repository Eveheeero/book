PMULUDQ—Multiply Packed Unsigned Doubleword Integers INSTRUCTION SET REFERENCE, M-U Vol. 2B 4-389 PMULUDQ—Multiply Packed Unsigned Doubleword Integers Instruction Operand Encoding Description Multiplies the first operand (destination operand) by the second operand (source operand) and stores the result in the destination operand. In 64-bit mode and not encoded with VEX/EVEX, using a REX prefix in the form of REX.R permits this instruction to access additional registers (XMM8-XMM15). Legacy SSE version 64-bit operand: The source operand can be an unsigned doubleword integer stored in the low doubleword of an MMX technology register or a 64-bit memory location. The destination operand can be an unsigned doubleword integer stored in the low doubleword an MMX technology register.
