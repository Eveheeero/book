C-2 Vol. 2D INTEL® C/C++ COMPILER INTRINSICS AND FUNCTIONAL EQUIVALENTS The “scalar” element is 1.0. Due to the nature of the instruction, some intrinsics require their arguments to be immediates (constant integer literals). To use an intrinsic in your code, insert a line with the following syntax: data_type intrinsic_name (parameters) Where: data_type Is the return data type, which can be either void, int, __m64, __m128, __m128d, or __m128i. Only the _mm_empty intrinsic returns void. intrinsic_name Is the name of the intrinsic, which behaves like a function that you can use in your C/C++ code instead of in-lining the actual instruction. parameters Represents the parameters required by each intrinsic.
