CPUID—CPU Identification INSTRUCTION SET REFERENCE, A-L 3-254 Vol. 2A How Brand Strings Work To use the brand string method, execute CPUID with EAX input of 8000002H through 80000004H. For each input value, CPUID returns 16 ASCII characters using EAX, EBX, ECX, and EDX. The returned string will be NULL-termi- nated.
