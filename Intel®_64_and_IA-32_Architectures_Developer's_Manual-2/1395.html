STD—Set Direction Flag INSTRUCTION SET REFERENCE, M-U Vol. 2B 4-661 STD—Set Direction Flag Instruction Operand Encoding Description Sets the DF flag in the EFLAGS register. When the DF flag is set to 1, string operations decrement the index regis- ters (ESI and/or EDI). Operation is the same in all modes. Operation DF := 1; Flags Affected The DF flag is set. The CF, OF, ZF, SF, AF, and PF flags are unaffected. Exceptions (All Operating Modes) #UD If the LOCK prefix is used.
