VMOVW—Move Word INSTRUCTION SET REFERENCE, V 5-406 Vol. 2C VMOVW—Move Word Instruction Operand Encoding Description This instruction either (a) copies one word element from an XMM register to a general-purpose register or memory location or (b) copies one word element from a general-purpose register or memory location to an XMM register. When writing a general-purpose register, the lower 16-bits of the register will contain the word value. The upper bits of the general-purpose register are written with zeros.
