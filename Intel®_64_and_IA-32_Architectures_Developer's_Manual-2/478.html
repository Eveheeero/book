FCHS—Change Sign INSTRUCTION SET REFERENCE, A-L 3-368 Vol. 2A FCHS—Change Sign Description Complements the sign bit of ST(0). This operation changes a positive value into a negative value of equal magni- tude or vice versa. The following table shows the results obtained when changing the sign of various classes of numbers.
