MOVUPS—Move Unaligned Packed Single Precision Floating-Point Values INSTRUCTION SET REFERENCE, M-U 4-132 Vol. 2B MOVUPS—Move Unaligned Packed Single Precision Floating-Point Values Instruction Operand Encoding Description Note: VEX.vvvv and EVEX.vvvv is reserved and must be 1111b otherwise instructions will #UD. EVEX.512 encoded version: Moves 512 bits of packed single precision floating-point values from the source operand (second operand) to the destination operand (first operand). This instruction can be used to load a ZMM register from a 512-bit float32 memory location, to store the contents of a ZMM register into memory. The destination operand is updated according to the writemask.
