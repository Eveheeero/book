VFCMULCSH/VFMULCSH—Complex Multiply Scalar FP16 Values INSTRUCTION SET REFERENCE, V 5-178 Vol. 2C VFCMULCSH/VFMULCSH—Complex Multiply Scalar FP16 Values Instruction Operand Encoding Description This instruction performs a complex multiply operation. There are normal and complex conjugate forms of the oper- ation. The masking for this operation is done on 32-bit quantities representing a pair of FP16 values. Bits 127:32 of the destination operand are copied from the corresponding bits of the first source operand. Bits MAXVL-1:128 of the destination operand are zeroed. The low FP16 element of the destination is updated according to the writemask. Rounding is performed at every FMA (fused multiply and add) boundary. Execution occurs as if all MXCSR excep- tions are masked. MXCSR status bits are updated to reflect exceptional conditions. Operation VFMULCSH dest{k1}, src1, src2 (AVX512) KL := VL / 32 IF k1[0] or *no writemask*: // non-conjugate version subtracts last even term tmp.fp16[0] := src1.fp16[0] * src2.fp16[0] tmp.fp16[1] := src1.fp16[1] * src2.fp16[0] dest.fp16[0] := tmp.fp16[0] - src1.fp16[1] * src2.fp16[1] dest.fp16[1] := tmp.fp16[1] + src1.fp16[0] * src2.fp16[1] ELSE IF *zeroing*: dest.fp16[0] := 0 dest.fp16[1] := 0 DEST[127:32] := src1[127:32] // copy upper part of src1 DEST[MAXVL-1:128] := 0 Opcode/ Instruction Op/ En 64/32 bit Mode Support CPUID Feature Flag Description EVEX.LLIG.F2.MAP6.W0 D7 /r VFCMULCSH xmm1{k1}{z}, xmm2, xmm3/m32 {er} A V/V AVX512-FP16 Complex multiply a pair of FP16 values from xmm2 and xmm3/m32, and store the result in xmm1 subject to writemask k1. Bits 127:32 of xmm2 are copied to xmm1[127:32].
