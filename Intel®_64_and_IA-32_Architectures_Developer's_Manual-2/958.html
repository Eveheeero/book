PAND—Logical AND INSTRUCTION SET REFERENCE, M-U 4-224 Vol. 2B PAND—Logical AND Instruction Operand Encoding Description Performs a bitwise logical AND operation on the first source operand and second source operand and stores the result in the destination operand. Each bit of the result is set to 1 if the corresponding bits of the first and second operands are 1, otherwise it is set to 0. In 64-bit mode and not encoded with VEX/EVEX, using a REX prefix in the form of REX.R permits this instruction to access additional registers (XMM8-XMM15).
