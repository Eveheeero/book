STC—Set Carry Flag INSTRUCTION SET REFERENCE, M-U 4-660 Vol. 2B STC—Set Carry Flag Instruction Operand Encoding Description Sets the CF flag in the EFLAGS register. Operation is the same in all modes. Operation CF := 1; Flags Affected The CF flag is set. The OF, ZF, SF, AF, and PF flags are unaffected.
