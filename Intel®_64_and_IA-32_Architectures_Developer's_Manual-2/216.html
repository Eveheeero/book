BLSMSK—Get Mask Up to Lowest Set Bit INSTRUCTION SET REFERENCE, A-L 3-106 Vol. 2A BLSMSK—Get Mask Up to Lowest Set Bit Instruction Operand Encoding Description Sets all the lower bits of the destination operand to “1” up to and including lowest set bit (=1) in the source operand. If source operand is zero, BLSMSK sets all bits of the destination operand to 1 and also sets CF to 1. This instruction is not supported in real mode and virtual-8086 mode. The operand size is always 32 bits if not in 64-bit mode. In 64-bit mode operand size 64 requires VEX.W1. VEX.W1 is ignored in non-64-bit modes. An attempt to execute this instruction with VEX.L not equal to 0 will cause #UD. Operation temp := (SRC-1) XOR (SRC) ; SF := temp[OperandSize -1]; ZF := 0; IF SRC = 0 CF := 1; ELSE CF := 0; FI DEST := temp; Flags Affected SF is updated based on the result. CF is set if the source if zero. ZF and OF flags are cleared. AF and PF flag are undefined.
