CMC—Complement Carry Flag INSTRUCTION SET REFERENCE, A-L 3-174 Vol. 2A CMC—Complement Carry Flag Instruction Operand Encoding Description Complements the CF flag in the EFLAGS register. CMC operation is the same in non-64-bit modes and 64-bit mode. Operation EFLAGS.CF[bit 0] := NOT EFLAGS.CF[bit 0]; Flags Affected The CF flag contains the complement of its original value. The OF, ZF, SF, AF, and PF flags are unaffected. Exceptions (All Operating Modes) #UD If the LOCK prefix is used.
