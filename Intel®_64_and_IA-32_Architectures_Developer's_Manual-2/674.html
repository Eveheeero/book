KANDNW/KANDNB/KANDNQ/KANDND—Bitwise Logical AND NOT Masks INSTRUCTION SET REFERENCE, A-L 3-564 Vol. 2A KANDNW/KANDNB/KANDNQ/KANDND—Bitwise Logical AND NOT Masks Instruction Operand Encoding Description Performs a bitwise AND NOT between the vector mask k2 and the vector mask k3, and writes the result into vector mask k1.
