KTESTW/KTESTB/KTESTQ/KTESTD—Packed Bit Test Masks and Set Flags INSTRUCTION SET REFERENCE, A-L Vol. 2A 3-575 KTESTW/KTESTB/KTESTQ/KTESTD—Packed Bit Test Masks and Set Flags Instruction Operand Encoding Description Performs a bitwise comparison of the bits of the first source operand and corresponding bits in the second source operand. If the AND operation produces all zeros, the ZF is set else the ZF is clear. If the bitwise AND operation of the inverted first source operand with the second source operand produces all zeros the CF is set else the CF is clear. Only the EFLAGS register is updated. Note: In VEX-encoded versions, VEX.vvvv is reserved and must be 1111b, otherwise instructions will #UD.
