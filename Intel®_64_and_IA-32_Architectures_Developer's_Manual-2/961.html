PANDN—Logical AND NOT INSTRUCTION SET REFERENCE, M-U Vol. 2B 4-227 PANDN—Logical AND NOT Instruction Operand Encoding Description Performs a bitwise logical NOT operation on the first source operand, then performs bitwise AND with second source operand and stores the result in the destination operand. Each bit of the result is set to 1 if the corre- sponding bit in the first operand is 0 and the corresponding bit in the second operand is 1, otherwise it is set to 0. In 64-bit mode and not encoded with VEX/EVEX, using a REX prefix in the form of REX.R permits this instruction to access additional registers (XMM8-XMM15).
