SBB—Integer Subtraction With Borrow INSTRUCTION SET REFERENCE, M-U Vol. 2B 4-601 SBB—Integer Subtraction With Borrow Instruction Operand Encoding Description Adds the source operand (second operand) and the carry (CF) flag, and subtracts the result from the destination operand (first operand). The result of the subtraction is stored in the destination operand. The destination operand can be a register or a memory location; the source operand can be an immediate, a register, or a memory location.
