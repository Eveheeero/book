ANDN—Logical AND NOT INSTRUCTION SET REFERENCE, A-L 3-80 Vol. 2A ANDN—Logical AND NOT Instruction Operand Encoding Description Performs a bitwise logical AND of inverted second operand (the first source operand) with the third operand (the second source operand). The result is stored in the first operand (destination operand). This instruction is not supported in real mode and virtual-8086 mode. The operand size is always 32 bits if not in 64-bit mode. In 64-bit mode operand size 64 requires VEX.W1. VEX.W1 is ignored in non-64-bit modes. An attempt to execute this instruction with VEX.L not equal to 0 will cause #UD. Operation DEST := (NOT SRC1) bitwiseAND SRC2; SF := DEST[OperandSize -1]; ZF := (DEST = 0); Flags Affected SF and ZF are updated based on result. OF and CF flags are cleared.
