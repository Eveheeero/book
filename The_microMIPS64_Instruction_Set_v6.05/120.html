The 5-bit signed offset is left shifted by two bits, zero-extended and then is added to the contents of GPR 29 to form the effective address. Restrictions: The effective address must be naturally-aligned. If either of the 2 least-significant bits of the address is non-zero, an Address Error exception occurs. Providing misaligned support for Release 6 is not a requirement for this instruction.
