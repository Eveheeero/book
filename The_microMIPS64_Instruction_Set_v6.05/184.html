COP2 ICoprocessor Operation to Coprocessor 2 MIPS® Architecture for Programmers Volume II-B: microMIPS64™ Instruction Set, Revision 6.05 184 Format: COP2 func microMIPS Purpose: Coprocessor Operation to Coprocessor 2 To perform an operation to Coprocessor 2. Description: CoprocessorOperation(2, cofun) An implementation-dependent operation is performed to Coprocessor 2, with the cofun value passed as an argument. The operation may specify and reference internal coprocessor registers, and may change the state of the coprocessor conditions, but does not modify state within the processor. Details of coprocessor operation and internal state are described in the documentation for each Coprocessor 2 implementation.
