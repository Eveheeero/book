SDC2 Store Doubleword from Coprocessor 2 386 MIPS® Architecture for Programmers Volume II-B: microMIPS64™ Instruction Set, Revision 6.05 Format: SDC2 rt, offset(base) microMIPS Purpose: Store Doubleword from Coprocessor 2 To store a doubleword from a Coprocessor 2 register to memory Description: memory[GPR[base] + offset]  CPR[2,rt,0] The 64-bit doubleword in Coprocessor 2 register rt is stored in memory at the location specified by the aligned effec- tive address. The 12-bit signed offset is added to the contents of GPR base to form the effective address. Restrictions: Pre-Release 6: An Address Error exception occurs if EffectiveAddress 2..0 ≠ 0 (not doubleword-aligned). Release 6 allows hardware to provide address misalignment support in lieu of requiring natural alignment. Note: The pseudocode is not completely adapted for Release 6 misalignment support as the handling is implementa- tion dependent.
