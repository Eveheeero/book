SB16 Store Byte (16-bit instr size) 109 MIPS® Architecture for Programmers Volume II-B: microMIPS64™ Instruction Set, Revision 6.05 Format: SB16 rt, offset(base) microMIPS Purpose: Store Byte (16-bit instr size) To store a byte to memory Description: memory[GPR[base]  offset]  GPR[rt] The least-significant 8-bit byte of GPR rt is stored in memory at the location specified by the effective address. The 4-bit unsigned offset is added to the contents of GPR base to form the effective address. Restrictions: The 3-bit base register field can only specify GPRs $2-$7, $16, $17. The 3-bit rt register field can only specify GPRs $0, $2-$7, $17.
