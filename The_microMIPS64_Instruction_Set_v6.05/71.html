ADDIUR2 Add Immediate Unsigned Word Two Registers (16-bit instr size) MIPS® Architecture for Programmers Volume II-B: microMIPS64™ Instruction Set, Revision 6.05 71 Programming Notes: The term “unsigned” in the instruction name is a misnomer; this operation is 32-bit modulo arithmetic that does not trap on overflow. This instruction is appropriate for unsigned arithmetic, such as address arithmetic, or integer arith- metic environments that ignore overflow, such as C language arithmetic.
