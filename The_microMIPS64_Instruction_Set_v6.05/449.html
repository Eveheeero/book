WRPGPR IWrite to GPR in Previous Shadow Set MIPS® Architecture for Programmers Volume II-B: microMIPS64™ Instruction Set, Revision 6.05 449 Format: WRPGPR rt, rs microMIPS Purpose: Write to GPR in Previous Shadow Set To move the contents of a current GPR to a GPR in the previous shadow set. Description: SGPR[SRSCtl PSS , rt]  GPR[rs] The contents of the current GPR rs is moved to the shadow GPR register specified by SRSCtl PSS (signifying the pre- vious shadow set number) and rt (specifying the register number within that set). Restrictions: In implementations prior to Release 2 of the Architecture, this instruction resulted in a Reserved Instruction excep- tion.
