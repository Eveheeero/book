SLT Set on Less Than 400 MIPS® Architecture for Programmers Volume II-B: microMIPS64™ Instruction Set, Revision 6.05 Format: SLT rd, rs, rt microMIPS Purpose: Set on Less Than To record the result of a less-than comparison. Description: GPR[rd]  (GPR[rs] < GPR[rt]) Compare the contents of GPR rs and GPR rt as signed integers; record the Boolean result of the comparison in GPR rd. If GPR rs is less than GPR rt, the result is 1 (true); otherwise, it is 0 (false). The arithmetic comparison does not cause an Integer Overflow exception.
