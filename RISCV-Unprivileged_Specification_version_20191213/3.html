Preface This document describes the RISC-V unprivileged architecture. The ISA modules marked Ratiﬁed have been ratiﬁed at this time. The modules marked Frozen are not expected to change signiﬁcantly before being put up for ratiﬁcation. The modules marked Draft are expected to change before ratiﬁcation.
