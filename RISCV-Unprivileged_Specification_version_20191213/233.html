Commit fence A fence instruction instance i in state Plain(Fence(kind, next state)) can be committed if: 1. if i is a normal fence and it has .pr set, all program-order-previous load instructions are ﬁnished; 2. if i is a normal fence and it has .pw set, all program-order-previous store instructions are ﬁnished; and 3. if i is a fence.tso, all program-order-previous load and store instructions are ﬁnished. Action: 1. record that i is committed; and 2. update the state of i to Plain(next state). Register read An instruction instance i in state Plain(Read reg(reg name, read cont)) can do a register read of reg name if every instruction instance that it needs to read from has already performed the expected reg name register write. Let read sources include, for each bit of reg name, the write to that bit by the most recent (in program order) instruction instance that can write to that bit, if any. If there is no such instruction, the source is the initial register value from initial register state. Let reg value be the value assembled from read sources. Action: 1. add reg name to i.reg reads with read sources and reg value; and 2. update the state of i to Plain(read cont(reg value)).
