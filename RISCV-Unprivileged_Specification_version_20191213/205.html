A.6.1 Possible Future Extensions We expect that any or all of the following possible future extensions would be compatible with the RVWMO memory model: • ‘V’ vector ISA extensions • A transactional memory subset of the ‘T’ ISA extension • ‘J’ JIT extension • Native encodings for load and store opcodes with aq and rl set • Fences limited to certain addresses • Cache writeback/ﬂush/invalidate/etc. instructions
