This concludes the informal description of the operational model. The following sections describe the formal operational model. B.3.1 Intra-instruction Pseudocode Execution The intra-instruction semantics for each instruction instance is expressed as a state machine, es- sentially running the instruction pseudocode. Given a pseudocode execution state, it computes the next state. Most states identify a pending memory or register operation, requested by the pseudocode, which the memory model has to do.
