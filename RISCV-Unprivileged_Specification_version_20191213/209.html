Appendix B Formal Memory Model Speciﬁcations, Version 0.1 To facilitate formal analysis of RVWMO, this chapter presents a set of formalizations using diﬀerent tools and modeling approaches. Any discrepancies are unintended; the expectation is that the models describe exactly the same sets of legal behaviors. This appendix should be treated as commentary; all normative material is provided in Chapter 14 and in the rest of the main body of the ISA speciﬁcation.
