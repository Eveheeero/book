Support for atomic instructions is divided into two categories: LR/SC and AMOs. Some platforms might mandate that all of cacheable main memory support all atomic operations required by the attached processors. 3.6.3.1 AMO PMA Within AMOs, there are four levels of support: AMONone, AMOSwap, AMOLogical, and AMOArithmetic. AMONone indicates that no AMO operations are supported. AMOSwap in- dicates that only amoswap instructions are supported in this address range. AMOLogical indi- cates that swap instructions plus all the logical AMOs (amoand, amoor, amoxor) are supported. AMOArithmetic indicates that all RISC-V AMOs are supported. For each level of support, natu- rally aligned AMOs of a given width are supported if the underlying memory region supports reads and writes of that width. Main memory and I/O regions may only support a subset or none of the processor-supported atomic operations.
