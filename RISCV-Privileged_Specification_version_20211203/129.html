When V=1, vsepc substitutes for the usual sepc, so instructions that normally read or modify sepc actually access vsepc instead. When V=0, vsepc does not directly aﬀect the behavior of the machine. vsepc is a WARL register that must be able to hold the same set of values that sepc can hold.
