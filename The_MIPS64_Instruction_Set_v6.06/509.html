TNEI ITrap if Not Equal Immediate The MIPS64® Instruction Set Reference Manual, Revision 6.06 564 Format: TNEI rs, immediate MIPS32, removed in Release 6 Purpose: Trap if Not Equal Immediate To compare a GPR to a constant and do a conditional trap. Description: if GPR[rs]  immediate then Trap Compare the contents of GPR rs and the 16-bit signed immediate as signed integers. If GPR rs is not equal to imme- diate, then take a Trap exception. Restrictions: None Availability and Compatibility: This instruction has been removed in Release 6.
