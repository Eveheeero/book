SRLV Shift Word Right Logical Variable 507 The MIPS64® Instruction Set Reference Manual, Revision 6.06 Format: SRLV rd, rt, rs MIPS32 Purpose: Shift Word Right Logical Variable To execute a logical right-shift of a word by a variable number of bits. Description: GPR[rd]  GPR[rt] >> GPR[rs] (logical) The contents of the low-order 32-bit word of GPR rt are shifted right, inserting zeros into the emptied bits; the word result is sign-extended and placed in GPR rd. The bit-shift amount is specified by the low-order 5 bits of GPR rs. Restrictions: On 64-bit processors, if GPR rt does not contain a sign-extended 32-bit value (bits 63..31 equal), then the result of the operation is UNPREDICTABLE.
