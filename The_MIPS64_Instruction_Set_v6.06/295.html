LUI Load Upper Immediate 321 The MIPS64® Instruction Set Reference Manual, Revision 6.06 Format: LUI rt, immediate MIPS32, Assembly Idiom Release 6 Purpose: Load Upper Immediate To load a constant into the upper half of a word Description: GPR[rt]  sign_extend(immediate || 0 16 ) The 16-bit immediate is shifted left 16 bits and concatenated with 16 bits of low-order zeros. The 32-bit result is sign- extended and placed into GPR rt. Restrictions: None. Operation: GPR[rt]  sign_extend(immediate || 0 16 ) Exceptions: None Programming Notes: In Release 6, LUI is an assembly idiom of AUI with rs=0.
