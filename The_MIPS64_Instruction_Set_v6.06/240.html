The instruction stream in the remote processor is synchronized with respect to the execution of GINVI once the SYNC operation following GINVI completes. The following sequence is recommended for use of GINVI. ginvi /* fully-invalidate all caches*/ sync 0x14 /* Enforce completion - all instruction streams synchronized. */ jr.hb ra /* Clear instruction hazards*/ Implementation Notes: None.
