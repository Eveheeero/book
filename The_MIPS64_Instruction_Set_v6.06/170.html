DADD IDoubleword Add The MIPS64® Instruction Set Reference Manual, Revision 6.06 185 Format: DADD rd, rs, rt MIPS64 Purpose: Doubleword Add To add 64-bit integers. If overflow occurs, then trap. Description: GPR[rd]  GPR[rs] + GPR[rt] The 64-bit doubleword value in GPR rt is added to the 64-bit value in GPR rs to produce a 64-bit result. If the addi- tion results in 64-bit 2’s complement arithmetic overflow, then the destination register is not modified and an Integer Overflow exception occurs. If it does not overflow, the 64-bit result is placed into GPR rd.
