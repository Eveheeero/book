SB IStore Byte The MIPS64® Instruction Set Reference Manual, Revision 6.06 448 Format: SB rt, offset(base) MIPS32 Purpose: Store Byte To store a byte to memory. Description: memory[GPR[base]  offset]  GPR[rt] The least-significant 8-bit byte of GPR rt is stored in memory at the location specified by the effective address. The 16-bit signed offset is added to the contents of GPR base to form the effective address.
