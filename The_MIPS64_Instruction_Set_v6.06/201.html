DMFC2 Doubleword Move from Coprocessor 2 218 The MIPS64® Instruction Set Reference Manual, Revision 6.06 Format: DMFC2 rt, rd MIPS64 DMFC2, rt, rd, sel MIPS64 The syntax shown above is an example using DMFC1 as a model. The specific syntax is implementation dependent. Purpose: Doubleword Move from Coprocessor 2 To move a doubleword from a coprocessor 2 register to a GPR. Description: GPR[rt]  CP2CPR[Impl] The contents of the coprocessor 2 register denoted by the Impl field is loaded into GPR rt. The interpretation of the Impl field is left entirely to the Coprocessor 2 implementation and is not specified by the architecture. Restrictions: The results are UNPREDICTABLE if Impl specifies a coprocessor 2 register that does not exist, or if the coproces- sor 2 register specified by rd and sel is a 32-bit register.
