DINS Doubleword Insert Bit Field 204 The MIPS64® Instruction Set Reference Manual, Revision 6.06 Exceptions: Reserved Instruction Programming Notes The assembler will accept any value of pos and size that satisfies the relationship 0  pos+size  64 and emit DINS, DINSM, or DINSU as appropriate to the values.
