If multiple levels of caches are to be affected by one CACHEE instruction, all of the affected cache levels must be processed in the same manner â either all affected cache levels use the globalized behavior or all affected cache levels use the non-globalized behavior. The CACHEE instruction functions the same as the CACHE instruction, except that address translation is performed using the user mode virtual address space mapping in the TLB when accessing an address within a memory segment configured to use the MUSUK access mode. Memory segments using UUSK or MUSK access modes are also acces- sible . Refer to Volume III, Enhanced Virtual Addressing section for additional information. Implementation of this instruction is specified by the Config5 EVA field being set to 1.
