TLTI ITrap if Less Than Immediate The MIPS64® Instruction Set Reference Manual, Revision 6.06 560 Format: TLTI rs, immediate MIPS32, removed in Release 6 Purpose: Trap if Less Than Immediate To compare a GPR to a constant and do a conditional trap. Description: if GPR[rs]  immediate then Trap Compare the contents of GPR rs and the 16-bit signed immediate as signed integers. If GPR rs is less than immediate, then take a Trap exception. Restrictions: None Availability and Compatibility: This instruction has been removed in Release 6.
