DIV MOD DIVU MODU DDIV DMOD DDIVU DMODU IDIV: Divide Words Signed MOD: Modulo Words The MIPS64® Instruction Set Reference Manual, Revision 6.06 214 Pre-Release 6 divide instructions that produce quotient and remainder in the HI/LO registers produce a Reserved Instruction exception on Release 6. In the future, the instruction encoding may be reused for other instructions. Programming Notes: Because the divide and modulo instructions are defined to not trap if dividing by zero, it is safe to emit code that checks for zero-divide after the divide or modulo instruction.
