ADDU Add Unsigned Word 58 The MIPS64® Instruction Set Reference Manual, Revision 6.06 Format: ADDU rd, rs, rt MIPS32 Purpose: Add Unsigned Word To add 32-bit integers. Description: GPR[rd]  GPR[rs] + GPR[rt] The 32-bit word value in GPR rt is added to the 32-bit value in GPR rs and the 32-bit arithmetic result is sign- extended and placed into GPR rd. No Integer Overflow exception occurs under any circumstances. Restrictions: If either GPR rt or GPR rs does not contain sign-extended 32-bit values (bits 63..31 equal), then the result of the operation is UNPREDICTABLE. Operation: if NotWordValue(GPR[rs]) or NotWordValue(GPR[rt]) then UNPREDICTABLE endif temp  GPR[rs] + GPR[rt] GPR[rd]  sign_extend(temp 31..0 ) Exceptions: None Programming Notes: The term “unsigned” in the instruction name is a misnomer; this operation is 32-bit modulo arithmetic that does not trap on overflow.
