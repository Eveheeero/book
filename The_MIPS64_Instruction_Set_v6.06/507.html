TLTU ITrap if Less Than Unsigned The MIPS64® Instruction Set Reference Manual, Revision 6.06 562 Format: TLTU rs, rt MIPS32 Purpose: Trap if Less Than Unsigned To compare GPRs and do a conditional trap. Description: if GPR[rs]  GPR[rt] then Trap Compare the contents of GPR rs and GPR rt as unsigned integers. If GPR rs is less than GPR rt, then take a Trap exception. The contents of the code field are ignored by hardware and may be used to encode information for system software. To retrieve the information, system software must load the instruction word from memory. Alternatively, if CP0 BadInstr is implemented, the code field may be obtained from BadInstr .
