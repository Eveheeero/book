MFHI Move From HI Register 363 The MIPS64® Instruction Set Reference Manual, Revision 6.06 Format: MFHI rd MIPS32, removed in Release 6 Purpose: Move From HI Register To copy the special purpose HI register to a GPR. Description: GPR[rd]  HI The contents of special register HI are loaded into GPR rd. Restrictions: Availability and Compatibility: This instruction has been removed in Release 6. Operation: GPR[rd]  HI Exceptions: None Historical Information: In the MIPS I, II, and III architectures, the two instructions which follow the MFHI must not modify the HI register. If this restriction is violated, the result of the MFHI is UNPREDICTABLE.
