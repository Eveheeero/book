MOVZ IMove Conditional on Zero The MIPS64® Instruction Set Reference Manual, Revision 6.06 376 Format: MOVZ rd, rs, rt MIPS32, removed in Release 6 Purpose: Move Conditional on Zero To conditionally move a GPR after testing a GPR value. Description: if GPR[rt] = 0 then GPR[rd]  GPR[rs] If the value in GPR rt is equal to zero, then the contents of GPR rs are placed into GPR rd. Restrictions: None Availability and Compatibility: This instruction has been removed in Release 6 and has been replaced by the ‘SELEQZ’ instruction. Refer to the SELEQZ instruction in this manual for more information.
