The RINT.fmt instruction corresponds to the roundToIntegralExact operation in the IEEE Standard for Floating- Point Arithmetic 754 TM -2008. The Inexact exception is signaled if the result does not have the same numerical value as the input operand. The floating point scalar instruction RINT.fmt corresponds to the MSA vector instruction FRINT.df. I.e. RINT.S cor- responds to FRINT.W, and RINT.D corresponds to FRINT.D. Availability and Compatibility: This instruction is introduced by and required as of Release 6.
