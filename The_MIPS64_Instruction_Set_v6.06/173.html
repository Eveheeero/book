DADDU Doubleword Add Unsigned 188 The MIPS64® Instruction Set Reference Manual, Revision 6.06 Format: DADDU rd, rs, rt MIPS64 Purpose: Doubleword Add Unsigned To add 64-bit integers Description: GPR[rd]  GPR[rs] + GPR[rt] The 64-bit doubleword value in GPR rt is added to the 64-bit value in GPR rs and the 64-bit arithmetic result is placed into GPR rd. No Integer Overflow exception occurs under any circumstances. Restrictions: Operation: GPR[rd]  GPR[rs] + GPR[rt] Exceptions: Reserved Instruction Programming Notes: The term “unsigned” in the instruction name is a misnomer; this operation is 64-bit modulo arithmetic that does not trap on overflow. It is appropriate for unsigned arithmetic such as address arithmetic, or integer arithmetic environ- ments that ignore overflow, such as C language arithmetic. 31 26 25 21 20 16 15 11 10 6 5 0 SPECIAL 000000 rs rt rd 0 00000 DADDU 101101 6 5 5 5 5 6
