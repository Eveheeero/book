DSUB Doubleword Subtract 240 The MIPS64® Instruction Set Reference Manual, Revision 6.06 Format: DSUB rd, rs, rt MIPS64 Purpose: Doubleword Subtract To subtract 64-bit integers; trap on overflow Description: GPR[rd]  GPR[rs] - GPR[rt] The 64-bit doubleword value in GPR rt is subtracted from the 64-bit value in GPR rs to produce a 64-bit result. If the subtraction results in 64-bit 2’s complement arithmetic overflow, then the destination register is not modified and an Integer Overflow exception occurs. If it does not overflow, the 64-bit result is placed into GPR rd.
