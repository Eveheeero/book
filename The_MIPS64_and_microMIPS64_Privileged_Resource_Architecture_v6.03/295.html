9.53 Load Linked Address (CP0 Register 17, Select 0) MIPS® Architecture For Programmers Volume III: MIPS64® / microMIPS64™ Privileged Resource Architecture, Rev. 6.03 295 9.53 Load Linked Address (CP0 Register 17, Select 0) Compliance Level: Optional prior to Release 5. Required in Release 5 if Config5 LLB =1. Required in Release 6. The LLAddr register contains relevant bits of the physical address read by the most recent Load Linked instruction. This register is implementation-dependent, is for diagnostic purposes only, and serves no function during normal operation. Release 5 also provides software with the ability to read and clear the LLbit, which is set when an LL instruction is executed. The presence of LLB in LLAddr in Release 5 can be detected by software through Config5 LLB . In Release 6, Config5 LLB is read-only 1, and CP0 LLAddr is required.
