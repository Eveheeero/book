Register State Value Cause WP Indicates that the watch exception was deferred until after both Status EXL and Status ERL were zero. This bit directly causes a watch exception, so software must clear this bit as part of the exception handler to prevent a watch exception loop at the end of the current handler execution. Register State Value Cause IP indicates the interrupts that are pending.
