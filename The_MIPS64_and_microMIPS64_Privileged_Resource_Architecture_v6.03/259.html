9.46 Configuration Register (CP0 Register 16, Select 0) MIPS® Architecture For Programmers Volume III: MIPS64® / microMIPS64™ Privileged Resource Architecture, Rev. 6.03 259 9.46 Configuration Register (CP0 Register 16, Select 0) Compliance Level: Required. The Config register specifies various configuration and capabilities information. Most of the fields in the Config reg- ister are initialized by hardware during the Reset Exception process, or are constant. Three fields, K23, KU, and K0, must be initialized by software in the reset exception handler. Figure 9.49 shows the format of the Config register; Table 9.67 describes the Config register fields.
