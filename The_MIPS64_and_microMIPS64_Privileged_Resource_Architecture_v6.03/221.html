9.33 Status Register (CP Register 12, Select 0) MIPS® Architecture For Programmers Volume III: MIPS64® / microMIPS64™ Privileged Resource Architecture, Rev. 6.03 221 UX 5 Enables the following behavior: • Access to 64-bit User Segments. • Use of the XTLB Refill Vector for references to User Segments. • Execution of instructions which perform 64-bit opera- tions while the processor is operating in User Mode. Prior to Release 5, or in Release 5 without Enhanced Vir- tual Addressing (EVA), access to 64-bit User Segment is disabled, when UX=0, by causing an Address Error. In Release 5 with EVA, and Release 6 onwards, access to 64- bit User Segment is prevented by explicitly sign-extending the address from bit 31 when KX=0 or SX=0 or UX=0, providing the inputs that generate the address are also sign-extended from bit 31. See “32-bit Compatibility Addressing for Release 5 EVA, and Release 6” on page 39 and Table 9.50 for further detail. In Release 5 with EVA, and Release 6 onwards, if KX or SX are written as 0, then UX is also written as 0. If 64-bit addressing is not implemented, this bit must be ignored on write and read as zero.
