9.43 CDMMBase Register (CP0 Register 15, Select 2) MIPS® Architecture For Programmers Volume III: MIPS64® / microMIPS64™ Privileged Resource Architecture, Rev. 6.03 253 9.43 CDMMBase Register (CP0 Register 15, Select 2) Compliance Level: Optional. The 64-bit physical base address for the Common Device Memory Map facility is defined by this register. This regis- ter only exists if Config3 CDMM is set to one. For devices that implement multiple VPEs, access to this register is controlled by the VPEConf0 MVP register field. If the MVP bit is cleared, a read to this register returns all zeros and a write to this register is ignored.
