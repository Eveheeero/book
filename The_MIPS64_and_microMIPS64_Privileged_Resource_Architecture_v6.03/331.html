9.65 CacheErr Register (CP0 Register 27, Select 0) MIPS® Architecture For Programmers Volume III: MIPS64® / microMIPS64™ Privileged Resource Architecture, Rev. 6.03 331 9.65 CacheErr Register (CP0 Register 27, Select 0) Compliance Level: Optional. The CacheErr register provides an interface with the cache error detection logic that may be implemented by a pro- cessor. The exact format of the CacheErr register is implementation-dependent and not specified by the architecture. Refer to the processor specification for the format of this register and a description of the fields.
