9.69 DataHi Register (CP0 Register 29, Select 1, 3) MIPS® Architecture For Programmers Volume III: MIPS64® / microMIPS64™ Privileged Resource Architecture, Rev. 6.03 339 9.69 DataHi Register (CP0 Register 29, Select 1, 3) Compliance Level: Optional. The DataLo and DataHi registers are registers that act as the interface to the cache data array and are intended for diagnostic operation only. The Index Load Tag operation of the CACHE instruction reads the corresponding data val- ues into the DataLo and DataHi registers. The exact format and operation of the DataLo and DataHi registers is implementation-dependent. Refer to the proces- sor specification for the format of this register and a description of the fields.
