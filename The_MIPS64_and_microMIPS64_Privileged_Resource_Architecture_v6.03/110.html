Additional State Saved Entry Vector Used General exception vector (offset 0x180) 6.2.13 TLB Invalid Exception A TLB invalid exception occurs when a TLB entry matches a reference to a mapped address space, but the matched entry has the valid bit off. Note that the condition in which no TLB entry matches a reference to a mapped address space and the EXL bit is one in the Status register is indistinguishable from a TLB Invalid Exception, in the sense that both use the general excep- tion vector and supply an ExcCode value of TLBL or TLBS. The only way to distinguish these two cases is by prob- ing the TLB for a matching entry (using TLBP).
