Figure 9.67 shows the format of the Performance Counter Counter Register; Table 9.86 describes the Performance Counter Coun- ter Register fields. K 1 Enables event counting in Kernel Mode. Unlike the usual definition of Kernel Mode as described in Section 3.2 “Kernel Mode” on page 23, this bit enables event counting only when the EXL and ERL bits in the Status register are zero. R/W Undefined Required EXL 0 Enables event counting when the EXL bit in the Status register is one and the ERL bit in the Status register is zero. Counting is never enabled when the ERL bit in the Status register or the DM bit in the Debug register is one.
