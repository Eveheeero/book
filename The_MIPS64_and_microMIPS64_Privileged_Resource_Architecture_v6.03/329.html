9.64 ErrCtl Register (CP0 Register 26, Select 0) MIPS® Architecture For Programmers Volume III: MIPS64® / microMIPS64™ Privileged Resource Architecture, Rev. 6.03 329 9.64 ErrCtl Register (CP0 Register 26, Select 0) Compliance Level: Optional. The ErrCtl register provides an implementation-dependent diagnostic interface with the error detection mechanisms implemented by the processor. This register has been used in previous implementations to read and write parity or ECC information to and from the primary or secondary cache data arrays in conjunction with specific encodings of the Cache instruction or other implementation-dependent method. The exact format of the ErrCtl register is imple- mentation-dependent and not specified by the architecture. Refer to the processor specification for the format of this register and a description of the fields.
