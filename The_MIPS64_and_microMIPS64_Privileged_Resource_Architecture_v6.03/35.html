4.7 Address Translation and Cacheability-and-Coherency Attributes for the xkphys Segment MIPS® Architecture For Programmers Volume III: MIPS64® / microMIPS64™ Privileged Resource Architecture, Rev. 6.03 35 4.7 Address Translation and Cacheability-and-Coherency Attributes for the xkphys Segment The xkphys Unmapped Segment comprises eight address ranges, each of which provides a window into the entire 2 PABITS bytes of physical memory; as such, it is not translated using the TLB or other address translation unit. For this Segment, the cacheability and coherency attribute is taken from VA 61..59 and has the same encoding as that shown in Table 9.11. An Address Error Exception occurs if VA 58..PABITS are non-zero. If no Address Error Exception occurs, the physical address is taken from the VA PABITS-1..0 virtual address field.
