9.33 Status Register (CP Register 12, Select 0) MIPS® Architecture For Programmers Volume III: MIPS64® / microMIPS64™ Privileged Resource Architecture, Rev. 6.03 213 9.33 Status Register (CP Register 12, Select 0) Compliance Level: Required. The Status register is a read/write register that contains the operating mode, interrupt enabling, and the diagnostic states of the processor. Fields of this register combine to create operating modes for the processor. See “MIPS64 and microMIPS64 Operating Modes” on page 23 for a discussion of operating modes, and “Interrupts” on page 85 for a discussion of interrupt modes. Figure 9.35 shows the format of the Status register for Pre-Release 6; Figure 9.35 shows the format of the Status reg- ister for Release 6; Table 9.49 describes the Status register fields. Release 6 modifies the behavior of Status KX/SX/UX . See Table 9.50 and the associated pseudocode for reference, in addition to Status KX/SX/UX .
