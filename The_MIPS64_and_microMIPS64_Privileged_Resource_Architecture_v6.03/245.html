9.40 Nested Exception Program Counter (CP0 Register 14, Select 2) MIPS® Architecture For Programmers Volume III: MIPS64® / microMIPS64™ Privileged Resource Architecture, Rev. 6.03 245 9.40 Nested Exception Program Counter (CP0 Register 14, Select 2) Compliance Level: Optional. The Nested Exception Program Counter (NestedEPC) is a read/write register with the same behavior as the EPC reg- ister except that: • The NestedEPC register ignores the value of Status EXL and is therefore updated on the occurrence of any excep- tion, including nested exceptions. • The NestedEPC register is not used by the ERET/DERET/IRET instructions. Software is required to copy the value of the NestedEPC register to the EPC register if it is desired to return to the address stored in NestedEPC. This register is part of the Nested Fault feature, existence of the register can be determined by reading the Config5 NFExists bit.
