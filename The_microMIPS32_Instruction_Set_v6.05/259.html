Description: CPR[0, rs, sel]  GPR[rt] The contents of general register rt are loaded into the coprocessor 0 register specified by the combination of rs and sel. Not all coprocessor 0 registers support the sel field. In those instances, the sel field must be set to zero. Restrictions: Pre-Release 6: The results are UNDEFINED if coprocessor 0 does not contain a register as specified by rs and sel. Release 6: Writes to a register that is reserved or not defined for the current core configuration are ignored.
