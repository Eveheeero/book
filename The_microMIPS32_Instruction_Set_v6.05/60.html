microMIPS Instruction Set 60 MIPS® Architecture for Programmers Volume II-B: microMIPS32™ Instruction Set, Revision 6.05 5.2 16-bit Instruction Register Set Many of the 16-bit instructions use 3-bit register specifiers in their binary encodings. The register set used for most of these 3-bit register specifiers is listed in Table 5.5. The register set used for SB16, SH16, SW16 source register is listed in Table 5.5. These register sets are a true subset of the register set available in 32-bit mode; the 3-bit register specifiers can directly access 8 of the 32 registers available in 32-bit mode (which uses 5-bit register specifiers). In addition, specific instructions in the 16-bit instruction set implicitly reference the stack pointer register (sp), global pointer register (gp), the return address register (ra), the integer multiplier/divider output registers (HI/LO) and the program counter (PC). Of these, Table 5.6 lists sp, gp and ra. Table 5.7 lists the microMIPS special-purpose registers, including PC, HI and LO. The microMIPS also contains some 16-bit instructions that use 5-bit register specifiers. Such 16-bit instructions pro- vide access to all 32 general-purpose registers.
