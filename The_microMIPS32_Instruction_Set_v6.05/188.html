For each of the div/mod operator pairs DIV/M OD, DIVU/MODU, the results satisfy the equation (A div B)*B + (A mod B) = A, where (A mod B) has same sign as the dividend A, and abs(A mod B) < abs(B). This equation uniquely defines the results. NOTE: if the divisor B=0, this equation cannot be satisfied, and the result is UNPREDICTABLE. This is commonly called “truncated division”. DIV performs a signed 32-bit integer division, and places the 32-bit quotient result in the destination register. MOD performs a signed 32-bit integer division, and places the 32-bit remainder result in the destination register. The remainder result has the same sign as the dividend. DIVU performs an unsigned 32-bit integer division, and places the 32-bit quotient result in the destination register. MODU performs an unsigned 32-bit integer division, and places the 32-bit remainder result in the destination regis- ter.
